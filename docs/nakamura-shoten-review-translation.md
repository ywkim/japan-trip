# 교토역 라멘코지(京都拉麺小路) — 10개 점포 비교

> 출처: 교토역 빌딩 공식(kyoto-station-building.co.jp/ramen_koji)·Kyotopi 2026 정리·Tabelog 2026-06-01 기준
>
> 6/2 점심 장소. 교토역 빌딩 **10층**에 전국 지역 라멘 **10개 점포**가 모인 푸드코트. 각 점포 앞 **자동 발권기(券売機)**에서 식권을 사고 자리에 앉는 방식이라 일본어가 어려워도 그림 메뉴로 주문 가능. 영업 **11:00~22:00(L.O.21:30)**.

## 추천 — 中村商店 (이번 점심 1순위)

라멘코지에서 **타베로그 최고점(3.56·리뷰 603, 2026-05 기준)**. 오사카 다카츠키에서 출발한 브랜드로, **시소(차조기) 향이 감도는 황금 소금(金の塩)** 과 닭을 오래 끓인 맑은 파이탄이 간판.

- **풍부한 닭 파이탄 간장 라멘 + 계란 ¥1,230** — 가장 많이 팔리는 조합
- 황금 소금 라멘 — 맑고 깔끔, **시부모님께 부담 없는 담백함**
- 미소·츠케멘 등 변형도 운영

폭우로 동산 코스를 접고 교토역 빌딩 안에서 점심을 해결하는 6/2 동선상, **닭파이탄 소금의 담백함**이 가족 4인 첫 끼로 무난합니다. 더 진한 걸 원하는 사람은 같은 층에서 도야마 블랙(이로하)·삿포로 미소(코테츠)를 각자 고르면 됩니다.

### 참고 — 라멘코지 인기 상위(타베로그 2026-05 리서치)
- 中村商店 **3.56** · 麺家いろは ~3.41 · 博多一幸舎 ~3.25
- (그 외 점포는 공식 라인업 기준 — 평점은 현장·타베로그에서 확인)

## 방문 팁

- **자동 발권기(券売機)**: 입장 전 발권기에서 식권 구매 후 착석. 그림·영어 병기 있음.
- **4인 좌석**: 각 점포는 카운터 위주로 작은 편(中村商店 19석). 연속 4석이 어려우면 **가족이 두 점포로 나눠 앉아 서로 다른 지역 라멘을 비교**하는 것도 라멘코지다운 방법.
- **혼잡**: 점심 피크(12~13시) 회피하려면 11:30 전 착석 권장. 6/2는 우천이라 실내 푸드코트가 더 붐빌 수 있음.
- **위치**: 교토역 빌딩 10층 — JR·지하철 교토역 직결, 카덴쇼에서 택시 단거리.
