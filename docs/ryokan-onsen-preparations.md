## 필수 준비물 (체크리스트)

### 1. 온천 기본용품 (료칸 제공 예상)

- [ ] **유카타 (浴衣)** — 료칸에서 객실당 제공. 목욕 후 입는 전통 면 옷. 앞을 왼쪽 넘기고 띠(`obi`)로 허리 졸라맴
  - 사용법 모르면 호텔 스태프에게 "ayumi-kata" (입는 방법) 요청
  - 좌우 구분 실수 있을 수 있으니 사진 촬영 또는 직원 지도 받기
- [ ] **목욕용 작은 타올** (wash towel) — 료칸 제공. 온천 입수 전 몸 씻을 때 사용 (일본 온천 에티켓)
- [ ] **큰 타올** (bath towel) — 료칸 제공. 목욕 후 몸 닦음
- [ ] **샴푸·린스·바디워시** — 료칸 제공 (개별 또는 고정식 디스펜서)
  - 개인용 필요 시 자체 준비 (민감한 두피·피부)

### 2. 개인용품 (자체 지참 권장)

- [ ] **슬리퍼/실내화** — 객실 복도·탕으로 이동할 때. 료칸이 제공하지만 편한 개인 슬리퍼 추천
- [ ] **수건용 헤어밴드 또는 헤어클립** — 긴 머리인 경우 목욕 시 머리 고정
- [ ] **데오도란트/제한스틱** (선택) — 온천 후 야외 활동 시
- [ ] **피부 민감제(민트 크림)** — 온천 후 피부 자극 시 (필요 시)
- [ ] **안경·렌즈 케이스** — 온천에서 벗은 안경 보관용 (미니 드라이백 추천)

### 3. 의류 (복장) 체크리스트

**카덴쇼 숙박 시 필요한 옷**:

#### 당일(6/2) 객실·숙소 내

- [ ] **편한 평상복** (반바지·무릎 위 반팔 또는 얇은 긴팔) — 체크인 후 객실 내 휴식용
- [ ] **숙소 공용 공간용 옷** (슬리퍼와 함께 이동 가능한 편한 옷)
- [ ] **저녁 식사용 옷** (가이세키 또는 식당 방문용 — 반바지나 편한 팬츠)
  - 가이세키·고급 레스토랑: 소박하지만 단정한 옷 (찢어진 옷·운동복 피하기)
  - 교토 식당은 보통 깔끔한 캐주얼 수용 (럭셔리 드레스 코드 드물음)

#### 온천·목욕 후

- [ ] **유카타 입은 후 외출용 외투** (선택) — 저녁 기온 떨어질 수 있음 (6월 초, 저녁 18~22℃)
  - 얇은 카디건 또는 가벼운 바람막이 추천
  - 유카타만으로는 쌀쌀할 수 있음
- [ ] **객실 복귀 후 편한 옷** (유카타 대신 개인 잠옷·편한 옷) — 밤중 재입욕 시 편의

#### 교토 5월말~6월초 날씨 기준

| 시간 | 온도 | 복장 가이드 |
|---|---|---|
| 낮 (09:00~17:00) | 23~27℃ | 반팔, 얇은 긴팔, 반바지, 얇은 바지 |
| 저녁 (18:00~22:00) | 18~22℃ | 긴팔 또는 얇은 외투 필수 |
| 새벽 (6/3 출국, 05:00~07:00) | 15~18℃ | 가벼운 자켓 또는 두꺼운 카디건 |

**주의**: 6월 초 (또는 장마 진입 시)에는 **비 대비 가벼운 우비/레인코트** 필수

#### 시부모 배려 (고령자 편의)

- [ ] **쉽게 벗고 입을 수 있는 옷** — 목욕 전후 복장 변경 시 활동성 우선
  - 지퍼·단추 많은 옷 피하기
  - 신발끈이 있는 신발 피하기 (슬리퍼·로퍼 선호)
  - 타이트한 진·레깅스 피하기 (목욕 후 미끄러운 피부에 입기 어려움)
- [ ] **목과 팔의 태양 보호 (오후 햇빛 강함)** — 자외선 차단 얇은 소매 내의 또는 UV 차단 팔토시
- [ ] **큰 사이즈 유카타 준비** — 숙소에서 요청 (혹시 제공 사이즈가 작으면)

### 4. 시부모 고령자 특별 준비물

**시부모(고령자) 안전·편의용**:

- [ ] **욕실용 논슬립 슬리퍼** — 미끄럼 방지. 료칸 바닥이 젖어 있을 수 있음
  - 개인용 가벼운 슬리퍼(EVA 소재) 추천
- [ ] **욕실용 손잡이/안전봉 확인** — 카덴쇼 도착 시 욕조·탕 손잡이 위치 재확인. 필요 시 호텔에 추가 도움 요청
- [ ] **물때 제거용 풋케어 스톤/수건** (선택) — 노년층 발 건조함 대비
- [ ] **보습 크림** — 온천 후 피부 건조함 대비 (특히 팔다리)
- [ ] **처방약(혈압제 등)** — 온천 후 약 복용 시간 확인. 온천으로 인한 혈압 변화 주의
- [ ] **간단 의료용품** — 밴드(상처 방지), 감기약, 소화제
  - 여행자보험 가입 확인 (4인 모두 필수)

## 온천·가족탕 에티켓 및 주의사항

### 한국과의 문화 차이: 알몸 에티켓

| 항목 | 한국 (찜질방·사우나) | 일본 온천 (공용탕) | 일본 온천 (가족탕/개인탕) |
|---|---|---|---|
| **기본 복장** | 알몸 당연함 (성인, 나이·성별 무관) | 알몸 당연함 (성인, 나이·성별 무관) | **알몸 당연함 (가족 내에서만)** |
| **성별 혼탕** | 없음 (남탕/여탕 분리) | 일부 온천 혼탕 있음 (드물지만) | 없음 (개인 예약탕이므로 가족만) |
| **어린이 동반** | 가능 (목욕탕 문화 발달) | 가능 (보통 미취학 어린이) | **가능 (가족만의 공간이므로 자유)** |
| **에티켓** | 수건 탕에 담그지 않기 (한국도 동일) | 수건 탕에 담그지 않기 + 세신 필수 | 가족끼리만 알몸 → 별도 에티켓 불필요 |

#### 카덴쇼 가족탕(貸切風呂)에서는?

✅ **가족 4인(부부 + 시부모)이 개인 예약탕 사용 → 알몸으로 목욕하는 것이 자연스럽고 당연함**

- 목욕은 알몸이 정상 (한국과 동일)
- 가족만의 공간이므로 외부인 노출 없음
- 유카타나 수건을 탕에 담그지 않는 것만 유의 (한국과 동일)

#### 혹시 공용 대욕장(大浴場)을 사용한다면?

⚠️ 공용탕은 성별·나이별로 구분:
- **여탕**: 시부모 할머니, 소연, 아내(있다면) 함께 입욕 가능
- **남탕**: 영욱, 시부 함께 입욕 가능
- 혼탕은 드물며, 있어도 선택사항

### 입욕 전 (準備)

1. **신발 제거** — 탕 입구(탈의실 경계)에서 신발·슬리퍼 벗기. 정해진 신발장에 배치
2. **소지품 정리** — 열쇠·휴대폰을 수건 주머니 또는 작은 바구니에 넣기
3. **작은 타올 준비** — "wash towel" 1장 들고 탕으로 이동 (일본식 관례)

### 입욕 중 (使用方法)

1. **먼저 세신 (「かけ湯」)**
   - 온천 입수 전 **작은 타올로 팔·다리·몸 전체 물 끼얹기** (온천 예의)
   - 타올을 온천에 담그지 않기 (음식물 오염 방지)
   - 물 온도 손으로 먼저 테스트 (보통 38~42℃, 뜨거울 수 있음)

2. **천천히 입수**
   - 시부모: 물 온도에 적응할 시간 충분히 가짐 (3~5분)
   - 혈압·심장 부담 방지: 갑자기 들어가지 않기
   - 물 온도가 너무 뜨거우면 현장 스태프에게 알리기

3. **입욕 시간** — 보통 10~20분. 너무 길지 않게
   - 시부모 장시간 입욕 피하기 (심장 부담)
   - 어지러우면 즉시 탕에서 나오기

4. **탕 내에서 금지 사항**
   - 수건/타올을 물에 담그기 금지
   - 비누·샴푸 탕 안에서 사용 금지 (세정실에서만)
   - 큰 소리로 말하기 금지 (다른 손님 배려)

### 입욕 후 (入浴後)

1. **세신 마무리** — 신선한 물로 마지막 한 번 더 헹굼 (선택)
2. **몸 닦기** — 큰 타올로 완전히 건조. 특히 발·발가락 사이 물기 제거 (곰팡이 방지)
3. **유카타 입기** — 호텔 제공 유카타 착용 (선택적. 방에 돌아가서 입어도 됨)
   - 유카타 여밈: **왼쪽** 앞자락을 먼저, **오른쪽** 뒤에서 감싸기 (반대면 상식이나 방문객 표시)
   - 띠(`obi`)로 허리 졸라매기

## 주의사항 & 체크포인트

### 시부모 안전

- [ ] **체크인 시** — 욕실 손잡이·경사 확인. 필요 시 추가 안전대 요청
- [ ] **저녁 식사 후** — 소화 시간 30분 후 입욕 (혈압 안정화 대기)
- [ ] **약 복용** — 온천 후 혈압제 등 약 복용 시간 사전 정하기
- [ ] **당일 체력** — 오후 도착·피로도 고려해 첫 입욕은 짧게 (5~10분)

### 기타

- [ ] **카메라/휴대폰** — 물에 빠뜨리지 않도록 락커에만 보관
- [ ] **타올·유카타 외 물품 온천 내부로 반출 금지** (료칸 규정)
- [ ] **객실 열쇠** — 탕으로 들고 가기. 객실·탕 입구 열쇠 분실 주의
- [ ] **시간 확인** — 공용탕 예약 또는 시간대 제약 있으면 미리 예약
