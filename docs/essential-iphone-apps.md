# 교토 여행 필수 아이폰 앱 가이드 (2026-05-28 리서치)

> **범위**: 2026-05-31~06-03 교토 4인 가족 여행을 위한 필수 앱 사전 설치·설정 가이드. 
> 각 앱별 용도·설치·사전 설정·현지 운영 방법을 정리.
> 
> **대상**: 영욱·소연·시부모 4인 (특히 시부모의 스마트폰 조작 부담을 줄이기 위해 미리 설치·테스트)

---

## 1. 필수 앱 5가지 (우선 설치 순서)

### 1.1 ICOCA (Apple Wallet) — 교통·편의점 결제

**용도**
- 버스·지하철·JR 개찰구 통과
- 편의점(세부·로손·패밀리마트) 결제
- 일부 자판기·택시 결제

**설치 방법**
- 별도 앱 설치 불필요 — iPhone 기본 제공 `Wallet` 앱 사용

**사전 설정**
- ✅ 5월 30일(토)까지 완료 필수
- 상세 절차: `docs/icoca-iphone-setup.md` 참조
- 4인 각자 ¥3,000 초기 충전

**현지 운영**
- Wallet 앱 → ICOCA 카드 탭 → 기계에 아이폰 상단 닿히기 (잠금해제 불필요)
- 충전 부족 시: 편의점 계산대에서 "イコカチャージお願いします" + 현금
- 잔액 확인: Wallet 앱의 카드 화면 상단

---

### 1.2 Google Maps — 길 찾기·맛집 검색

**용도**
- 버스·지하철·도보 길 찾기 + 소요시간
- 주변 식당·편의점·관광지 검색
- 오프라인 지도 다운로드 (지하철 진입 시)

**설치 방법**
- App Store 검색 → `Google Maps` (무료) → `받기`

**사전 설정** (5월 30일까지)
1. 앱 열기 → 좌측 상단 프로필 아이콘 → `오프라인 지도` 클릭
2. `+ 지도 선택` → 교토·오사카 영역 드래그 → `저장`
   - 저장 크기: 약 200MB ~ 500MB (WiFi 환경에서만)
3. 다운로드 완료 후 지하철·산 지역에서 데이터 없이도 사용 가능

**소연의 저장 목록 연계**
- 소연이 미리 구글맵에 저장한 41개 장소(`docs/soyeon-maps-list.md` 참조)를 4인이 공유하려면:
  - 구글 계정으로 로그인 → 좌측 메뉴 `내 지도` → 소연 공유 지도 확인
  - 모바일에서 확인하고 현지에서 "가는 길" 클릭하면 자동 길 찾기

**현지 운영**
- 버스/지하철 탈 때: `길 찾기` → 출발지·도착지 입력 → 버스/지하철 아이콘 선택 → "지금 출발"
- 맛집 찾을 때: 검색창에 음식 종류 + "근처" 입력 (예: "우동 근처") → 평점순 정렬
- 시부모가 운영할 때: 화면 크기 키우기 (설정 → 디스플레이·텍스트 크기 → 더 크게)

---

### 1.3 Tabelog(タベログ) — 맛집 평가·리뷰

**용도**
- 일본 최대 맛집 평점 사이트 (Michelin·가이드북보다 실명 리뷰 풍부)
- 요리 사진·메뉴·가격대·혼잡 시간 확인
- 예약 링크 직연결 (식べログ 예약·Tabelog 앱 예약)

**설치 방법**
- App Store 검색 → `Tabelog` (무료) → `받기`
- 또는 웹브라우저 → `tabelog.com` (모바일 최적화)

**사전 설정** (선택)
- 회원가입 불필수 (비로그인 사용 가능)
- Google/Facebook 계정으로 빠르게 회원가입 가능
- 로그인 시: 즐겨찾기·리뷰 저장 가능 (다음 방문 시 히스토리 유지)

**현지 운영**
- 앱 검색창 → 음식 종류 또는 지역명 입력 (예: "교토역 카레" or "히가시야마 회전초밥")
- 평점순 정렬 (상단 필터) → 3.5점 이상 추천
- 예약 링크 있으면 "예약"→ 리불렛(r.gnavi.co.jp) 또는 식べログ 자체 예약 페이지로 이동
- 찍은 사진으로 식재료 알레르기·금지식 확인 (시부모의 식이 제약 확인용)

**참고**: `data/itinerary.json`의 일자별 식사는 Tabelog 평점 기준으로 선정됨. 현지에서 추가 발굴할 때 참고.

---

### 1.4 Papago (라인 번역) — 일본어↔한국어 번역

**용도**
- 메뉴판·간판 촬영 후 한국어 번역
- 가게 직원과의 대화 실시간 번역
- 긴 문장 번역 (Google 번역보다 한국어 문맥이 자연스러움)

**설치 방법**
- 옵션 A: App Store `Papago` (Line Corporation, 무료) → `받기`
- 옵션 B: Google Play Store `Google 번역` 사용 (iOS에서도 가능)

**사전 설정**
- 두 앱 다 설치 권장 (한쪽이 부족할 때 다른 쪽 사용)
- Papago: 회원가입 선택. 비로그인도 기본 기능 사용 가능
- 오프라인 번역팩 다운로드:
  - Papago 앱 → 메뉴 → `오프라인 번역` → 일본어팩 다운로드 (약 50MB)
  - 인터넷 없어도 텍스트 번역 가능

**현지 운영**
1. **메뉴판 촬영**: Papago 앱 → `카메라` 탭 → 메뉴판 촬영 → 자동 번역
2. **음성 번역** (선택): 앱 내 `음성` 탭 → 마이크 아이콘 → 일본어로 말하기 → 한국어 텍스트 출력
3. **텍스트 입력**: 검색창에 일본어 입력 → 한국어로 번역 확인

**주의**: 욕설·의료·법률용어는 자동 번역이 부정확할 수 있음. 가게 직원과 중요한 내용은 2차 확인 추천.

---

### 1.5 Trip.com (또는 Booking.com) — 예약 확인·호텔 정보

**용도**
- ICOCA 충전 위치(JR 역사 매표소·편의점) 찾기
- 호텔·숙박·항공 예약 번호 확인
- 현지 관광 정보·가이드 투어 예약

**설치 방법**
- App Store 검색 → `Trip.com` (무료) 또는 `Booking.com` (무료)

**사전 설정** (5월 30일까지)
- 회원가입/로그인 → 예약 내역 불러오기
- 여행 일정 등록 (자동으로 예약을 기반 스케줄 구성)
- 홈화면에 예약 카드 고정 (호텔·항공 번호 빠르게 확인)

**현지 운영**
- 호텔 도착 시: 예약 내역 제시 (체크인 가속화)
- 그 외 식당/관광지 추가 예약 필요 시: 앱 내 "예약" 탭에서 현지 숙박·가이드 투어 검색·예약

## 4. 현지 운영 흐름도

### 4.1 오전 (호텔 → 첫 목적지)

1. Google Maps → "길 찾기" → 버스/지하철 선택
2. 버스·지하철 탈 때 → ICOCA 아이폰 탭 (Wallet 앱 미니 카드)
3. 목적지 도착 → Tabelog로 근처 맛집 검색 (점심)
4. 메뉴판 읽기 어려움 → Papago로 촬영·번역

### 4.2 점심 시간

1. 예약 경험 없으면 → Tabelog "예약" 링크 또는 가게 직접 전화 (일본어 불가 시 → Papago 음성)
2. 가게 찾기 어려움 → Google Maps에서 식당 주소 복사 → 새 길 찾기

### 4.3 오후 (관광·쇼핑)

1. 관광지 정보 → Google Maps 평점·영업시간 확인
2. 편의점 결제 → ICOCA (카운터에서 이코카 충전 가능)
3. 지하철 산 지역 진입 시 → 오프라인 Google Maps 지도 사용 (데이터 절약)

### 4.4 저녁 (저녁 식사·귀숙)

1. 저녁 식당 → Tabelog로 인기순 검색
2. 호텔 도착 → Trip.com의 숙박 카드에서 체크인 정보 재확인

## 6. 트러블슈팅

### 6.1 ICOCA가 인식 안 될 때

- iPhone 상단 NFC 칩 영역(카메라 근처)을 기계에 닿히기
- 혹은 기계 상단(카드 삽입구) 주변 시도
- 그래도 안 되면 → `docs/icoca-iphone-setup.md §4` 참조

### 6.2 앱 이름을 못 찾을 때

- App Store 검색에서 영문명 입력 (예: `Google Maps` 대신 로마자)
- 또는 본 문서의 링크 복사 → App Store에서 직접 열기

### 6.3 시부모가 앱 삭제했을 때

- 5월 30일까지: 다시 설치 → 설정
- 현지(5/31~6/3): 영욱·소연 폰에서 공유 기능 사용 또는 호텔 WiFi에서 재설치
