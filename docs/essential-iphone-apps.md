# 교토 여행 필수 아이폰 앱 가이드 (2026-05-28 리서치)

> **범위**: 2026-05-31~06-03 교토 4인 가족 여행을 위한 필수 앱 사전 설치·설정 가이드. 
> **대상**: 영욱·소연·시부모 4인 (특히 시부모의 스마트폰 조작 부담을 줄이기 위해 미리 설치·테스트)
>
> 절대 필수 2개 + 강력 권장 2개로 구성. 과도한 설치 권장 제외.

---

## 1. 절대 필수 2가지

### 1.1 ICOCA (Apple Wallet) — 교통·편의점 결제 [필수]

**용도**
- 버스·지하철·JR 개찰구 통과 (교토 이동의 90%)
- 편의점(세브·로손·패밀리마트) 결제
- 자판기·택시 결제

**설치 방법**
- 별도 앱 설치 불필요 — iPhone 기본 제공 `Wallet` 앱 사용

**사전 설정** (5월 30일까지 필수)
- 상세 절차: `docs/icoca-iphone-setup.md` 참조
- 4인 각자 ¥3,000 초기 충전 + Express Card 설정

**현지 운영**
- Wallet 앱 → ICOCA 카드 탭 → 기계에 아이폰 상단 닿히기 (잠금해제 불필요)
- 충전 부족 시: 편의점 계산대에서 "イコカチャージお願いします" + 현금

---

### 1.2 Google Maps — 길 찾기 [필수]

**용도**
- 버스·지하철·도보 길 찾기 + 소요시간 (모든 이동의 기본)
- 주변 식당·편의점·관광지 검색
- 오프라인 지도 다운로드 (지하철 진입 시 데이터 절약)

**설치 방법**
- [Google Maps 다운로드 (App Store)](https://apps.apple.com/kr/app/google-maps/id585027354) — 무료

**사전 설정** (5월 30일까지)
1. 앱 열기 → 좌측 상단 프로필 아이콘 → `오프라인 지도` 
2. `+ 지도 선택` → 교토·오사카 영역 드래그 → `저장` (약 200MB ~ 500MB, WiFi 필수)
3. 오프라인 지도로 지하철·산 지역에서 데이터 없이도 길 찾기 가능

**현지 운영**
- 버스/지하철 탈 때: `길 찾기` → 출발지·도착지 → 버스/지하철 아이콘 → "지금 출발"
- 맛집 찾을 때: 검색창에 "우동 근처" 같이 입력 → 평점순 정렬

---

## 2. 강력 권장 2가지

### 2.1 Tabelog — 맛집 평가·리뷰 [권장]

**용도**
- 일본 최대 맛집 평점 사이트 (Google Maps보다 상세한 리뷰·사진·메뉴)
- 예약 링크 직연결

**설치 방법**
- [Tabelog 다운로드 (App Store)](https://apps.apple.com/kr/app/tabelog/id1199035773) — 무료
- 또는 웹브라우저 `tabelog.com`

**현지 운영**
- 앱 검색 → 음식 종류 + 지역명 입력 (예: "교토역 카레") → 3.5점 이상 추천
- Google Maps의 리뷰보다 실명 리뷰가 풍부 (예약 가능 여부도 즉시 확인)

---

### 2.2 Papago — 일본어↔한국어 번역 [권장]

**용도**
- 메뉴판·간판 촬영 후 한국어 번역
- 가게 직원과의 대화 실시간 번역

**설치 방법**
- [Papago 다운로드 (App Store)](https://apps.apple.com/kr/app/papago/id1147308408) — Line Corporation, 무료
- 또는 [Google 번역 다운로드](https://apps.apple.com/kr/app/google-translate/id414706506) — 무료 (둘 다 설치 권장)

**사전 설정**
- Papago 앱 → 메뉴 → `오프라인 번역` → 일본어팩 다운로드 (약 50MB)
- 인터넷 없어도 텍스트 번역 가능

**현지 운영**
- 메뉴판 촬영: Papago 앱 → `카메라` 탭 → 촬영 → 자동 번역
- 텍스트 입력: 일본어 입력 → 한국어로 번역 확인

## 5. 현지 운영 흐름도

### 오전 (호텔 → 첫 목적지)
1. Google Maps: 길 찾기 → 버스/지하철 선택
2. ICOCA: 아이폰 상단으로 개찰구 결제

### 점심
1. Tabelog: 근처 맛집 검색 (평점순)
2. Papago: 메뉴판 촬영 번역 (필요 시)
3. ICOCA: 편의점 결제 또는 캐시리스 카드 사용

### 오후 (관광·쇼핑)
1. Google Maps: 관광지 정보·평점 확인
2. 편의점 내 WiFi로 데이터 충전 (필요 시)

### 저녁 (호텔 귀숙)
1. Google Maps: 저녁 식당 길 찾기
2. 호텔 WiFi로 모든 앱 업데이트
