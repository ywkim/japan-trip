# 후보지 상세 비교

> 동반 인원: **부부 2인 + 시부모 2인 = 4인** (고령자 2인 포함). 모든 항목은 4인·시부모 동반 전제로 해석.

각 후보의 특징·이동·계절·식사·관광 정보를 함께 채우는 공간.

## 도쿄 (관동)

- **이동**: 인천 → 나리타/하네다, 약 2.5시간
- **계절**: 5~6월 초여름, 9~10월 가을
- **식사**: 다양한 선택지, 한국어 메뉴 보편
- **관광**: 시내 관광·박물관·쇼핑 중심
- **숙소**: 신주쿠/시부야/긴자, 호텔·료칸 다양
- **의료**: 한국어 가능 병원 다수 (성루카·아키야마 등)
- **시부모 적합성**: GMP-HND 직항 가능(이동 부담 최소). 단 도시 규모가 커 관광 동선이 길어질 수 있음. 호텔 위치(긴자·신주쿠)에 따라 보행량 조절 필요
