# 후보지 상세 비교

각 후보의 특징·이동·계절·식사·관광 정보를 함께 채우는 공간.

## 도쿄 (관동)

- **이동**: 인천 → 나리타/하네다, 약 2.5시간
- **계절**: 5~6월 초여름, 9~10월 가을
- **식사**: 다양한 선택지, 한국어 메뉴 보편
- **관광**: 시내 관광·박물관·쇼핑 중심
- **숙소**: 신주쿠/시부야/긴자, 호텔·료칸 다양
- **의료**: 한국어 가능 병원 다수 (성루카·아키야마 등)
- **메모**:

## 오사카 (관서)

- **이동**: 인천 → 간사이, 약 2시간
- **계절**: 5~6월 온난, 9~10월 쾌적
- **식사**: 다코야키·오코노미야키 등 미식
- **관광**: 도톤보리·USJ·오사카성
- **숙소**: 난바/우메다
- **의료**: 한국어 가능 병원 있음
- **메모**:

## 교토 (관서)

- **이동**: 인천 → 간사이 → 교토 (전철 약 1시간)
- **계절**: 5월 신록, 9~10월 단풍 직전
- **식사**: 전통 가이세키, 두부 요리
- **관광**: 사찰·정원·기온 거리 (도보 많음)
- **숙소**: 료칸 경험 가능
- **의료**: 일반 병원, 한국어 제한적
- **메모**:

## 삿포로 (홋카이도)

- **이동**: 인천 → 신치토세, 약 3시간
- **계절**: 5~6월 서늘, 9~10월 단풍·빠른 추위
- **식사**: 해산물·라멘·징기스칸
- **관광**: 자연·온천·드라이브
- **숙소**: 시내 호텔·온천 료칸
- **의료**: 시내 병원, 한국어 제한적
