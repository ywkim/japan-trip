# 제주항공 인천-고베 신규 취항 (2026-06)

> **요약**: 제주항공(7C)이 2026-06-11부터 인천(ICN) ↔ 고베(UKB) 직항을 매일 운항. 취항 기념 프로모션 진행. 본토 장마 시즌(6월)의 정량 의사결정 대안으로 고베를 후보지에 추가.
>
> 사용자 발화에서 "제주공항" 으로 표현되었으나 실제 신규 노선 주체는 **제주항공(Jeju Air, 항공사)** 의 인천-고베 노선 — 2026-05-11 확인 후 영욱 정정.

## 1. 노선 개요

| 항목 | 내용 |
|---|---|
| 항공사 | 제주항공 (Jeju Air, IATA: 7C) |
| 구간 | 인천국제공항(ICN) ↔ 고베공항(UKB) |
| 취항일 | 2026-06-11 |
| 빈도 | **매일 운항** |
| 인천 출발 | 13:35 → 고베 도착 15:15 (현지) |

> 제주항공은 인천-고베 노선 신설로 일본 간사이권 진입 게이트웨이를 오사카(간사이공항, KIX)에서 고베(UKB)로도 확장. 고베공항은 시내 접근성이 간사이공항보다 빠르다(포트라이너 18분, 산노미야).

## 4. 의사결정 기준에 미치는 영향

### 4.1 비용 (`cost`)
- **6월 한정 한시적 이점**: 신규 취항 프로모션 기간(~5/24 구매)에 한해 J포인트·쿠폰 가산. 6월 출발 시 실효 절감 약 ₩30,000~₩60,000 수준.
- 정량적으로 큰 차이는 아니나, **장마 페널티가 큰 6월에 본토 선택지를 살려두는 의미**가 더 큼.

### 4.2 신체 부담 (`physical_burden`)
- 고베공항 → 산노미야 **포트라이너 18분** (간사이공항 → 시내 60분+ 대비 절감)
- 짐·시부모 동반 시 환승 부담↓
- 단, 6월은 장마 영향권 — 강수일 13일/월 추정 (`data/weather.json` 참조)

### 4.3 경험 가치 (`experience`)
- 고베규(神戸牛) 본고장
- 키타노이진칸(이국적 양관 거리), 산노미야·모토마치 시가지
- 아리마 온천 당일치기 가능 (시내에서 전철+버스 약 45분)
- 메리켄파크·하버랜드 야경 (포트타워)
- 6월은 아지사이(수국) 시즌 — 모리야마 가든·롯코산

### 4.4 가족 적합성 (`family_fit`)
- 시내 평지·좁은 동선 → 교토(도보·계단) 대비 시부모 부담↓
- 호텔 선택지 풍부, 한식 식당 일부 있음

### 4.5 계절성 (`seasonality`)
- **2026-06: 5점** (장마 영향. 해안 입지로 교토 분지 4점보다 한 단계 양호)
- 2026-05·10: 9점 (가장 쾌적)
- 2026-09: 5점 (잔서·태풍 잔재)
