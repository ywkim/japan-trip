# 3M 예산 비교 도구

> 영욱·소연 부부의 일본 여행 예산 상한 **₩3,000,000 (시부모 동반 4인 합계)** 을 하드캡으로 보고
> 항공·숙박·고정비·일회성 비용 옵션을 카탈로그화하여 시나리오별로 통과 여부를 판정한다.

## 사용

```bash
python3 scripts/budget.py
```

각 시나리오에 대해:
- ✓ **확정 항목**: 실제 검색·공식 운임으로 확인된 가격
- · **미확정(TBD) 항목**: 매물·식당에 따라 변동, `data/cost-options.json`에 실측치 입력 필요
- ✅/❌ **3M 통과 여부**: 확정만으로 이미 초과되면 시나리오 자체 불가
- → **TBD 합계 상한**: 통과를 위해 TBD 항목들 합계가 이 금액 이하여야 한다는 가이드

## 데이터 구조 (`data/cost-options.json`)

| 섹션 | 의미 | 예 |
|---|---|---|
| `budget_cap_krw` | 예산 상한 | 3,000,000 |
| `trip_pax` | 여행자 수 | 4 |
| `flights` | 항공편 후보 (날짜·박수 포함) | ICN↔KIX, ICN↔NGS |
| `lodging` | 숙박 후보 (박당 단가) | 료칸·에어비앤비·시내 호텔 |
| `daily_fixed` | 일별 고정비 (1인1일 단가) | 식사 |
| `one_time` | 일회성 비용 | 공항 교통, 입장권 |
| `scenarios` | 위 옵션의 조합 | 교토 추천안, 나가사키·HTB |

### 가격 입력 규칙

- **`source`와 `data_quality` 필드 필수**
  - `confirmed_booking`: 실제 검색·예약 사이트 스크린샷 가격
  - `official_fare`: JR·시 교통 등 공식 운임
  - `researched_market_rate`: 비교 사이트·블로그·여행 가이드 시세 (예약 시 갱신 필요)
- **데이터 품질 추정 금지** — 실제 매물·운임·리서치된 시세만 입력
- 시나리오는 위 항목 id를 참조만 하므로, 단가 갱신 시 시나리오는 자동 반영

## 시나리오 추가 방법

```json
{
  "id": "my_scenario",
  "label": "내 시나리오 설명",
  "flight": "rs_kix_may24_27",
  "lodging_plan": [
    {"lodging": "kyoto_airbnb_4pax", "nights": 2},
    {"lodging": "kadensho_no_breakfast", "nights": 1}
  ],
  "fixed_days": [
    {"daily_fixed": "food_kyoto", "days": 4}
  ],
  "one_time": ["haruka_rt_4pax", "kyoto_local_3day_4pax", "kyoto_temples_4pax"]
}
```

## 현재 상태 (2026-05-09 리서치 반영)

### 환율 가정

100엔 ≈ 950원 (2026-05 시점)

### 데이터 품질 분류

- `confirmed_booking` (실측 예약 가격): 항공 2건, 카덴쇼 료칸 2건, HTB 입장권
- `official_fare` (공식 운임): JR·시 교통·사찰 입장료·교토 숙박세
- `researched_market_rate` (리서치 시세): 료칸·에어비앤비·시내 호텔·식사·군함도·사세보 교통

### 핵심 인사이트

1. **카덴쇼 본관(₩1.6M/박)은 3M 하드캡과 양립 불가** — 다른 항목 모두 절약해도 초과
2. **카덴쇼 5월 평일 최저가(₩502K/박)이면 3M 통과 가능** — Klook 화요일 최저 ₩250K/실 × 2 기준
3. **마츠바야 료칸(₩350K/박)이 가성비 최고** — 료칸 1박 경험 + 3M 안 양립
4. **나가사키·HTB도 가능** — 단 사세보 비즈니스호텔 + 고속버스 + 절약 식비 조합
5. **Forest Villa(HTB 내) + JR + 중간 식비**는 약 ₩200K 초과 — 식비 절약 또는 사세보 시내 호텔로 ₩200K 절감 시 통과

### 시나리오 6·7을 통과시키려면

- 시나리오 6 (나가사키 Forest Villa): 식비 중간형(₩28.5K)→절약형(₩16.5K) 시 ₩240K 절감 → ₩2,963K로 통과
- 시나리오 7 (카덴쇼 조식 포함): 카덴쇼 자체가 ₩1.6M라 통과 어려움. 자매관(아라시야마 카덴쇼 ₩550K)으로 변경 시 ₩1M 절감 가능

## 정책

- **3M은 하드캡** — 1원이라도 초과하는 시나리오는 채택 불가
- **TBD를 추측치로 채우지 않음** — 실제 매물·메뉴 가격만 입력
- 시나리오 ID는 카탈로그에 누적, 폐기된 시나리오도 보존 (변경 이력)
