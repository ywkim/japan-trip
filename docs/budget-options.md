# 3M 예산 비교 도구

> 영욱·소연 부부의 일본 여행 예산 상한 **₩3,000,000 (시부모 동반 4인 합계)** 을 하드캡으로 보고
> 항공·숙박·고정비·일회성 비용 옵션을 카탈로그화하여 시나리오별로 통과 여부를 판정한다.

## 사용

```bash
python3 scripts/budget.py
```

각 시나리오에 대해:
- ✓ **확정 항목**: 실제 검색·공식 운임으로 확인된 가격
- · **미확정(TBD) 항목**: 매물·식당에 따라 변동, `data/cost-options.json`에 실측치 입력 필요
- ✅/❌ **3M 통과 여부**: 확정만으로 이미 초과되면 시나리오 자체 불가
- → **TBD 합계 상한**: 통과를 위해 TBD 항목들 합계가 이 금액 이하여야 한다는 가이드

## 데이터 구조 (`data/cost-options.json`)

| 섹션 | 의미 | 예 |
|---|---|---|
| `budget_cap_krw` | 예산 상한 | 3,000,000 |
| `trip_pax` | 여행자 수 | 4 |
| `flights` | 항공편 후보 (날짜·박수 포함) | ICN↔KIX, ICN↔NGS |
| `lodging` | 숙박 후보 (박당 단가) | 료칸·에어비앤비·시내 호텔 |
| `daily_fixed` | 일별 고정비 (1인1일 단가) | 식사 |
| `one_time` | 일회성 비용 | 공항 교통, 입장권 |
| `scenarios` | 위 옵션의 조합 | 교토 추천안, 나가사키·HTB |

### 가격 입력 규칙

- **확정 가격은 `source` 필드와 함께 입력** (예: "Skyscanner 2026-05-09")
- **미확정은 `null` 유지** — 절대 추측치로 채우지 않음 (정확도 ↑ 후 갱신)
- 시나리오는 위 항목 id를 참조만 하므로, 단가 갱신 시 시나리오는 자동 반영

## 시나리오 추가 방법

```json
{
  "id": "my_scenario",
  "label": "내 시나리오 설명",
  "flight": "rs_kix_may24_27",
  "lodging_plan": [
    {"lodging": "kyoto_airbnb_4pax", "nights": 2},
    {"lodging": "kadensho_no_breakfast", "nights": 1}
  ],
  "fixed_days": [
    {"daily_fixed": "food_kyoto", "days": 4}
  ],
  "one_time": ["haruka_rt_4pax", "kyoto_local_3day_4pax", "kyoto_temples_4pax"]
}
```

### 미확정 (실측 필요)

- 교토 에어비앤비 (4인 1채) — Airbnb 검색 필요
- 교토 시내 호텔 (4인 2실) — 호텔 사이트 검색 필요
- 나가사키 시내 호텔 (4인 2실) — 호텔 사이트 검색 필요
- HTB 내 호텔 (4인) — HTB 공식 검색 필요
- 식사 1인1일 평균 (교토·나가사키) — 식당 메뉴 조사 필요
- 나가사키 공항-시내 리무진 왕복 — 공식 운임 확인
- JR 나가사키↔사세보 왕복 — JR 공식 확인
- 군함도 크루즈 — 운영사 가격 확인

## 시나리오 평가 결과 요약 (확정만 기준)

| 시나리오 | 확정 소계 | 3M 대비 | TBD 항목 |
|---|---|---|---|
| 교토 추천안 (료칸 조식 포함) | ₩3,170,564 | **❌ 초과 ₩170K** | 에어비앤비 2박 + 식사 |
| 교토 최저가안 (료칸 조식 미포함) | ₩2,801,902 | ✅ 여유 ₩198K | 에어비앤비 2박 + 식사 |
| 교토 료칸 제외안 (전 박 에어비앤비) | ₩1,570,638 | ✅ 여유 ₩1,429K | 에어비앤비 3박 + 식사 |
| 나가사키·HTB 4박5일 | ₩1,385,825 | ✅ 여유 ₩1,614K | 호텔 4박 + 식사 + 교통·크루즈 |

### 핵심 인사이트

1. **료칸 조식 포함안은 확정만으로 이미 3M 초과** — 다른 항목 0원이어도 통과 불가
2. **료칸 조식 미포함안도 여유 ₩198K뿐** — 에어비앤비 2박과 식사 4인 4일 합계가 그 안이어야 함
   - 산술적으로: 에어비앤비 박당 ≤ ₩50K · 식사 1인1일 ≤ ₩6K 등 비현실적
3. **료칸을 빼면 여유 ₩1.4M** — 현실적 가능 범위
4. **나가사키·HTB는 여유 ₩1.6M** — 4박 호텔·식사·교통·크루즈가 그 안이어야 함

## 정책

- **3M은 하드캡** — 1원이라도 초과하는 시나리오는 채택 불가
- **TBD를 추측치로 채우지 않음** — 실제 매물·메뉴 가격만 입력
- 시나리오 ID는 카탈로그에 누적, 폐기된 시나리오도 보존 (변경 이력)
