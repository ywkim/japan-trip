## 왜 がんこ(간코)는 뺐나

같은 요도바시 6F의 **간코(がんこ)**도 후보였지만, 일본어 후기 확인 결과 **초밥이 실제로 도는 게 아니라 터치패널 주문→레인 배달** 방식이고(회전 아님), 평점이 갈리며(타베로그 3.01·리뷰 4건, "조리스시 ¥2,000 가치 못 느낌" 혹평 포함) 약간 고가라 제외했다. "진짜 도는 회전초밥"이라는 기준에는 위 두 곳이 맞다.

## 권장

- **싸게 + 테이블에 둘러앉아 + 진짜 회전** → **くら寿司**([타베로그 3.03/43](https://tabelog.com/kyoto/A2601/A260604/26013042/), ¥115균일·1인 ¥1,000~1,200, 박스석, 택시 5분·요금 저렴). 4인 총액이 ①②보다 ₩1.5~2.5만 싸고 시부모가 테이블에 편히 앉는다. 단 **맛은 저가 체인 수준**.
- **맛·평판·교토 노포 정통 우위** → **무사시**([타베로그 3.42/255](https://tabelog.com/en/kyoto/A2601/A260101/26003088/), 1977 노포, 장인). 단 카운터 34석·상시 줄·하치조구치라 **4인+시부모는 대기 각오**.
- **4인 좌석 여유·산지직송 신선도·즉석 주문·동선** → **다이키수산**([타베로그 3.07/89](https://tabelog.com/kyoto/A2601/A260101/26028875/), 카운터 54석, ¥110~, 가라스마구치라 카덴쇼서 가까움). **폭우 저녁 4인+시부모 동선엔 이쪽이 무난**해 일정 기본값으로 두었다.
- 요약: **가격·좌석 우선이면 くら寿司**, **맛 우선이면 무사시**, **신선도+역앞 동선 절충이면 다이키수산**. ①② 두 역앞 집은 카운터석만이라 시부모가 테이블에 둘러앉는 그림은 ③ くら寿司가 유일하다.
