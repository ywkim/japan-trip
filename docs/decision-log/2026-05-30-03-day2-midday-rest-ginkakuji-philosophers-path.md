# 2026-05-30 — 6/1 둘째날 오후 재구성: 한낮 휴식 + 은각사·철학의 길 + 길 위 카페

## Status

Accepted

## Context (왜)

- 사용자 지시: "둘째날 점심 먹고 한낮은 에어비앤비 있다가 해 좀 지고 다시 나오려고 하는데 일정 조정 좀." 30도 더위·시부모(고령자 2인) 동반이라 가장 더운 한낮(13~15시)을 실외 관광으로 보내는 기존 6/1 오후(금각사 13:30 → 료안지 15:00)가 부담.
- 기존 6/1 오후는 북서(금각사·료안지) 사찰 2곳을 시버스 3회로 도는 동선이었다. 두 사찰 모두 17:00 마감이라 "한낮 휴식 후 해 지고 외출"과 양립 불가(은각사·금각사·료안지 등 사찰은 일몰 ~19시 전에 닫힘).
- 사용자가 대안을 직접 제시: "은각사 가고 철학의 길 가는 코스는 어때 금각사 빼고." 추가로 "중간에 카페 리서치해서 가는 길에 중간에 꼭 쉬게 넣어" — 더위 대피용 카페 휴식 슬롯 명시.
- 리서치 확정 사실(2026-05-30):
  - 은각사(銀閣寺): 여름(3~11월) 08:30~17:00·마지막 입장 17:00·¥500·관람 ~30~40분 (쇼코쿠지 공식).
  - 철학의 길(哲学の道): 은각사~난젠지 약 2km·천천히 30~45분, 비와호 운하변 나무 그늘길 (Japan Guide).
  - 니조역↔케아게역: 지하철 도자이선 환승 없이 직결(둘 다 도자이선)·~13분·¥260. 케아게→난젠지 도보 5~10분(네지리만포 터널).
  - 요지야카페 은각사점: 철학의 길 *중간*·일본정원·냉방·09:15~18:00.
  - 은각사도→시조가와라마치: 시버스 17/203 균일 ¥230·~35분.
- 알려진 대안과 한계: ① 금각사·료안지를 16시 이후로 미루기 → 17:00 마감이라 한 곳도 빠듯, 둘 다는 불가. ② 둘 다 생략 후 완전 휴식 → 사용자가 은각사+철학의 길을 원함. ③ 금각사를 다른 날로 이동 → 6/2가 이미 빡빡.

## Decision (무엇)

- 6/1(둘째날) 오후를 **점심 후 시오 복귀 → 한낮 휴식 → (지하철 직결) 케아게·난젠지 → 철학의 길 북상 → 요지야카페(길 중간 휴식) → 은각사 → 시버스로 가와라마치 복귀 → 폰토초 만자라테이 저녁**으로 재구성한다.
- **금각사·료안지(북서)는 6/1 확정 코스에서 제외**한다(정반대편이라 동쪽 히가시야마 집중과 양립 불가). 단 `route_candidates`의 "서북 사찰 집중형" 대안에는 그대로 보존.
- 동선 방향은 **케아게(난젠지)부터 철학의 길을 북상**하도록 잡는다. 은각사 우선(북→남)은 니조에서 지하철 2회 환승+버스가 필요해 시부모 동반 환승 부담이 크고, 케아게 직결(환승 0)이 가장 단순하기 때문. 은각사는 16:40 입장으로 17:00 마감 전 여유 확보.
- 카페는 **요지야카페 은각사점**(철학의 길 중간·정원·냉방)으로 "가는 길에 꼭 쉬는" 슬롯을 충족.

## Consequences (그래서)

- 긍정: 가장 더운 한낮을 실내(시오)에서 회피 + 철학의 길이 늦은 오후 그늘 산책이 됨. 동선이 동쪽 히가시야마 한 곳에 모여 이동·환승이 줄고(니조→케아게 환승 0), 카페 휴식으로 고령자 체력 배분. "해 지고 다시 나오기" 취지 충족.
- 부정·트레이드오프: 금각사(킨카쿠지)·료안지를 이번 여행 확정 코스에서 못 봄. 아라시야마(서)→시오(중)→케아게(동)로 도시를 횡단하는 큰 이동이 한 번 생김(한낮 휴식의 대가). 은각사는 16:40 입장이라 마감 전 ~35분 관람으로 다소 타이트.
- 후속 행동: 일부 leg(지하철 ¥260, 시버스 17/203 ~35분)는 `researched_market_rate` — 실제 탑승 전 구글맵/공식 시각표로 재확인 권장. 요지야카페 좌석 대기 시 블루보틀 교토(난젠지·09:00~18:00)로 대체 가능.
