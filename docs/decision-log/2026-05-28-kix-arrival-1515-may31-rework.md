# 2026-05-28 — KIX 15:15 도착 정정에 따른 5/31 일정 재구성

## Status

Accepted

## Context (왜)

- `data/booking-checklist.json`의 확정 발권은 에어서울 A8YW58 **ICN 13:15 → KIX 15:15** (귀국 KIX 10:05 → ICN 12:05, 2026-05-12 시부 PC 결제 확정).
- 그러나 `data/itinerary.json`의 5/31 일정은 **09:05 KIX 도착**(시부 예약 항공편 가정) 기준으로 짜여 있었음. 마크다운 사본(`docs/kyoto-itinerary-may31-jun3-2026.md`) §3에 "두 출처 단일화 필요 — 별도 PR"로 플래그된 상태.
- 09:05 도착 가정 위에 쌓인 5/31 낮 일정(11:30 짐 보관·12:00 니조성·12:45 점심 大鵬·13:30 키요미즈데라·16:00 맛차하우스·16:30 산넨자카/야사카/기온·17:00 철학의 길·18:30 가와라마치 저녁)은 실제 15:15 도착 기준으로는 **모두 도착 전 시간대**라 불가능.
- 실제 5/31은 KIX 15:15 도착 → 입국·수하물 ~16:15 → 하루카 75분 → 교토역 ~17:45 → 산인선 + 도보 → 시오 체크인 ~18:15. 도착·이동만으로 저녁이 되는 빡빡한 일정.
- 사용자 확인: "5월 31일 일정이 좀 이상한데? 간사이 공항 도착이 15시 15분인데" — 도착 시각 오류를 명시적으로 지적.
- 재구성 방향은 두 질문(저녁 시간 / 낮 관광 재배치)에 대해 "합리적으로 판단"을 위임받음.
- 동산 지구(키요미즈·기온·산넨자카·맛차하우스) 재배치 검토: 6/1(아라시야마 + 사이호지 10:30 예약 확정 + 금각/료안 northwest)·6/2(후시미이나리 + 토후쿠지 + 카덴쇼 13:30 체크인)은 이미 패킹된 상태에서 시부모 동반(고령자 2인). 키요미즈(17:00 마감)는 6/2 오전 후시미+토후쿠지 사이에 끼울 수 있으나 4시간 만에 동남쪽 3 사찰을 강행하는 셈이라 고령자 부담 큼. 강행 대신 본 PR에선 동산 주간 코스를 제외하고, 기온·야사카는 폰토초 저녁 인접 도보권이라 식후 야경 산책으로만 선택 편입.

## Decision (무엇)

- **정본을 `booking-checklist.json` 확정 발권(KIX 15:15)으로 통일**하고 `data/itinerary.json` 5/31을 다음으로 재구성:
  - 08:30 아침식사 (집)
  - 09:30 집 출발 → 인천공항 T1 (체크인 카운터 10:45 오픈, 3시간 전 도착 권장)
  - 15:15 KIX 도착 → 입국·수하물 → JR 하루카
  - 18:15 니조역 → 시오 체크인·짐 정리 (하루카 75분 + 산인선 + 도보 포함)
  - 19:00 가와라마치 저녁 — 폰토초 豆八 (지하철 도자이선 1회 이동, 호리고타쓰 완전개실)
  - 20:30 기온·야사카 신사 야경 산책 (여력 시, 폰토초 도보 0.8km)
- **5/31 동산 주간 코스 6개 항목 제거**: 니조성 외정원·中華 大鵬 점심·키요미즈데라·MACCHA HOUSE·산넨자카/야사카(주간)·철학의 길 단축 구간.
- **저녁 폰토초 豆八 (food_quality researched_market_rate 유지)**: 시간 18:30→19:00, arrive_from 모드 bus(철학의 길→가와라마치)→subway(니조→폰토초)로 재작성.
- **trip 레벨 메타 동시 정정**: `walking_km_total` 13→10, `transit_pass_playbook`의 "5/31 시오→키요미즈" 엔트리를 "5/31 시오→폰토초 저녁"으로 교체, 5/31 `pass_recommendation` 시버스 2회 ¥1,840→지하철 1회 ¥880, `pending` 4 항목 정정·제거(항공편 시간 정합성·tbd 라벨 leg 목록·니조성 본궁·철학의 길 대안).
- **route_candidates 3개(relaxed·northwest_temples·food_culture) 5/31 모두 동일하게 "오후 도착 + 체크인 + 저녁" 3 항목으로 단축**, 각 후보의 `walking_km_total`을 1km씩 차감(9→8, 14→12, 12→10).
- **사람용 마크다운 사본(`docs/kyoto-itinerary-may31-jun3-2026.md`) 동기화**: §1.1 5/31 표 재작성, §2.1 동산 블로그 링크 정리(폰토초/야사카/기온만 유지), §2.5 교통패스 표 5/31 행 정정 + 합계 ¥6,320→¥5,360, §3 정합성 항목을 "해결됨"으로 변경, §4 후보 코스 5/31 행 단축.
- **채택하지 않은 대안**:
  - 키요미즈를 6/2 오전에 강행 — 후시미이나리(7:30) + 토후쿠지(9:30) + 키요미즈(11:00) + 카덴쇼 체크인(13:30) 4시간 내 3 사찰. 고령자 동반·체력·이동 부담으로 기각.
  - 5/31 저녁을 시오 인근에서만 마무리 — 폰토초/가와라마치 야간 분위기 + 호리고타쓰 완전개실(시부모 좌식 편함) 가치 큼. 지하철 1회(¥220×4)로 접근 가능해 채택.
  - 동산 블로그 후기 카드를 본문 그대로 유지 — 도착 시각상 방문 불가능한 장소를 일정 카드에 두면 시각 화면에 잘못된 신호. 일정에서 제거, 일지·git history에 보존.

## Consequences (그래서)

- **긍정**:
  - itinerary.json과 booking-checklist.json의 출도착 시각이 단일 출처로 통일. `docs/kyoto-itinerary-may31-jun3-2026.md` §3에 두 차례(2026-05-19 PR #?, 2026-05-24 booking-research) 플래그된 정합성 보류 항목 해결.
  - 5/31이 도착·이동·체크인 + 저녁 + 선택적 야경 산책으로 명료해져 시부모 동반 첫날 동선이 무리 없음.
  - 교통패스 비용 ¥6,320(시버스 5회 기준)→¥5,360(지하철 1회 + 6/1 시버스 + 6/2 JR)으로 감소. trip.transit_pass_source의 ¥5,400 라벨과 ¥40 차로 거의 일치.
