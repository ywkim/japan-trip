## 핵심 관찰

- **예산 절감**: 카덴쇼 1박 ¥39,230 (¥76,500 → ¥37,270) ≈ ₩372,685 절감. 시나리오 합계 ₩3.42M → 약 ₩3.05M (5M 옵션 내 여유 확대).
- **부분 동기화 잔존**: itinerary.json(line 23, 477), docs/kyoto-itinerary-may31-jun3-2026.md(line 8), 2026-05-13 결정 일지(line 25)는 **이미 "조식 미포함"으로 정확히 기록돼 있었음** — 즉, 5/13 예약 시점에 일정 파일은 사람이 동기화했으나 비용/요약 파일은 누락된 채로 남아 있었던 것.
- **schema 한계**: booking-checklist.json은 reference·confirmed_at 별도 필드가 없어 note 단일 필드에 모두 embed하는 기존 패턴 유지(flight A8YW58과 동일 처리). 별도 필드 분리는 schema·validator·build·tests 동반 변경 필요 → 별도 PR 권장.
- **아카이브 처리**: final-report.md는 결정 시점(2026-05-12) 분석 결과로 보존, 본문 수치 직접 수정 대신 상단 사후 메모로 차이를 명시(이력 보존 + 정확성 양립).
