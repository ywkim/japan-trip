# 2026-05-28 — 간사이 오후 도착 확정에 따른 5/31 재구성·동산 코스 6/2 이동

## Status

Accepted

## Context (왜)

- 에어서울 A8YW58(시부 발권, `data/booking-checklist.json` 항공 항목 확정·발권 완료)의 실제 시각이 **ICN 13:15 → KIX 15:15 / 복로 KIX 10:05 → ICN 12:05**임이 확정.
- 그동안 `data/itinerary.json`의 5/31 도착은 오전 `09:05` 플레이스홀더(`시부 예약 항공편 시간 확인 필요`)였고, 6/3 출국편도 시각 미기입. `docs/kyoto-itinerary-may31-jun3-2026.md` §3에 두 출처 단일화가 보류로 남아 있었다.
- 오전 09:05 도착을 전제로 5/31에 니조성 외정원·키요미즈데라·맛차하우스·산넨자카·기온·철학의 길까지 빡빡하게 배치돼 있었으나, **15:15 오후 도착**이면 입국·수하물(45~60분)·하루카(75분)·환승을 거쳐 시오 체크인이 **17:45경**이라 첫날 관광이 물리적으로 불가능.
- 한일 시차 없음(둘 다 UTC+9)이라 13:15→15:15는 2시간(정상 비행시간)으로 정합.
- 사용자 확인(2026-05-28): ① 5/31 저녁은 이동 부담 없는 **시오 도보권 大鵬(쓰촨 중식)**, ② 5/31에서 빠지는 **니조성 외정원·철학의 길은 일정에서 제거**, ③ 동산 코스(키요미즈·산넨자카·맛차하우스·기온)는 **6/2 오후 카덴쇼 짐 위탁 후** 일괄 추가.
- 大鵬 일요일 디너 영업(17:30~21:00, L.O.20:30·화·격주 수 휴무)·주소(西ノ京星池町149)·전화(075-822-5598)는 Tabelog·Hot Pepper 2026-05-28 재확인.

## Decision (무엇)

- **5/31을 "오후 도착 + 시오 체크인 + 가까운 저녁"으로 축소**: 15:15 KIX 도착 → 17:45 시오 체크인 → 18:30 大鵬(원래 12:45 점심 항목을 저녁으로 승격). 니조성 외정원·철학의 길·폰토초 豆八(저녁) 항목 삭제.
- **동산 코스를 6/2 오후로 이동**: 키요미즈데라(13:00)·맛차하우스(14:30)·산넨자카·야사카·기온(15:15)을 카덴쇼 조기 짐 위탁(12:00) 후 배치. 료칸 대욕장 17:15·가이세키 18:00은 저녁 복귀 후 유지. 별도 드럭스토어 항목은 기온 메모로 흡수.
- **6/3 출국편 확정 반영**: KIX 10:05 출발 기준 06:15 체크아웃·06:45 하루카 탑승(국제선 2시간+ 버퍼).
- 대안(채택 안 함): 5/31 저녁을 폰토초 豆八로 유지 — 도착편 피로·이동 부담으로 기각. 동산 코스를 6/1에 끼워넣기 — 6/1(아라시야마+사이호지)이 이미 가득.
- `route_candidates`(대안 코스 3개)는 5/31 도착 시각만 확정값으로 갱신하고 이후 시각은 오후 도착 반영 전 스케치로 명시(정본은 `days`).
