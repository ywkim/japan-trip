# 2026-05-25 — 6/2 일정 재구성 + 네이버 후기 25개 일괄 추가

## Status

Accepted

## Context (왜)

- 6/1 사이호지 풀 일정 편입(PR #47)으로 6/2는 휴식 페이스가 필요. 기존 6/2(07:30 후시미이나리 → 09:30 토후쿠지 → 11:30 라멘코지 → 13:30 카덴쇼)는 시부모 동반에 새벽 출발 부담 + 카덴쇼 짐 처리 불명확.
- 사용자 결정 (2026-05-25 세션 내):
  - 시오 조식: 편의점/자취 → **시오 근처 카페** (구글맵 저장 리스트 — 코메다 등)로 변경 ("편의점을 시부모님이랑 어떻게 먹어")
  - 짐 처리: 시오 체크아웃 후 **카덴쇼 짐 위탁 먼저**, 그 다음 관광
  - 후시미이나리: 도리이 터널만 1시간 단축 코스 유지
  - 토후쿠지: 한번 빼기로 했다가 사용자가 "토후쿠지 좋다고" 재포함
  - 교토 철도박물관: 추가 제안했으나 사용자가 "철도박물관 싫다고" 거절
- 사용자 명시 요청: 새/변경 항목에 **네이버 후기 5개씩 링크 추가**. 자동화 시도:
  - WebSearch `site:blog.naver.com` 결과 0개 (검색 API 제약)
  - WebFetch `search.naver.com` 차단
  - 해결: **m.search.naver.com 모바일 검색을 curl + iPhone UA로 호출** → 정규식으로 `m.blog.naver.com/<user>/<postid>` 추출 → 각 포스트의 `og:image`·`og:description` 메타 추출
  - 5개 키워드 × 5 URL = 25개 후기를 자동 수집 성공

## Decision (무엇)

`data/itinerary.json`의 6/2 일정을 사용자 결정대로 재배치하고, 변경/추가 항목에 네이버 후기 25개를 자동 수집·삽입한다.

### 네이버 후기 자동 수집 방법

- m.search.naver.com 모바일 검색 (iPhone Safari UA)을 curl로 호출
- 정규식 `m\.blog\.naver\.com/[a-zA-Z0-9_-]+/[0-9]+` 으로 포스트 URL 추출, `awk '!seen[$0]++'` 으로 중복 제거
- 각 포스트 URL을 fetch → `og:image`·`og:description` 메타 정규식 추출
- 5개 키워드 (후시미이나리 천본도리이 후기 / 교토역 라멘코지 후기 / 토후쿠지 정원 후기 / 우메코지 카덴쇼 료칸 후기 / 교토 코메다 니조 모닝 후기) × 5 URL = **25개 entries** 완성
- 추출 성공률: 25/25 (모든 항목 img+comment 보유)
- 코멘트 80자 이내 트리밍

### 채택하지 않은 대안

- **WebSearch site:blog.naver.com**: 검색 결과 0개. 폐기.
- **WebFetch search.naver.com**: Claude Code 도메인 차단. 폐기.
- **검색 URL 그대로 blog_reviews에 넣기**: 사용자가 한번 더 클릭해야 함, 실제 후기와 다른 UX. 폐기.
- **교토 철도박물관 추가**: 사용자 거절.
- **토후쿠지 패스**: 사용자 재포함 결정.

## Consequences (그래서)

**긍정**
- 6/1(서쪽 풀 일정) → 6/2(시오 조식 + 후시미·토후쿠지 + 료칸) 페이스 차이 명확. 시부모 동반 페이스 조절.
- 카덴쇼 짐 위탁이 일정 카드에 명시 → 사용자 운영 부담 ↓.
- 변경 항목 5개에 각 5개씩 실제 네이버 포스트 + og:image 보유 → viz/itinerary 풍부.
- 네이버 후기 자동 수집 파이프라인 검증 (curl + iPhone UA → m.search.naver.com → og 추출) — 향후 재사용 가능.

**부정·트레이드오프**
- 카덴쇼 얼리 짐 위탁(09:30)은 호텔 정책 의존 — 출발 전 전화 확인 필요 (+81-75-316-0055).
- 토후쿠지 6월은 단풍 시즌이 아니라 신록만 — 매니악한 정원 미감이 시부모 호응 낮을 수 있음 (사용자 결정 존중).
- 자동 수집 후기는 검수 없이 삽입 — 후기 내용이 본인 시점과 다를 수 있음. 출발 전 사용자 검토 권장.
- 시오 근처 조식 가게는 itinerary에 "코메다 등" 통칭으로만 — 당일 가족 합의로 선택.

**후속 행동**
- 카덴쇼 전화 확인 (얼리 짐 위탁 + 가능하면 13:00 객실 입실).
- 출발 전 후기 25개 빠르게 훑어보고 부적합 후기 제거.
- 6/1 사이호지(PR #47)·요시무라에도 동일 방식으로 5개씩 후기 추가 (별도 PR).
