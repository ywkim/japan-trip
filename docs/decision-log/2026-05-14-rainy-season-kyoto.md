# 2026-05-14 — 교토 5/31~6/3 장마 진입 정량 분석

- 합의:
  - 교토 5/31~6/3 일정은 **평년상 매우입(6/6) 직전 진입기**로, 본격 장마 회피 확률 ~86% (최근 7년 실적 기준)
  - 본격 강수는 6월 중순(65.6mm/10일)·하순(89.7mm/10일)부터 — 여행 4일은 평년 곡선상 평탄 직전 구간
  - 우천 대비는 "여행 전체가 장마"보다 "단발성 우천 대응"으로 설계
- 보류:
  - 우천 시 실내 백업 동선(플랜 B) 별도 PR — 키요미즈데라·아라시야마 우천 대안 식별
  - 우천 준비물 체크리스트(우산·우비·드라이백) — 별도 PR
- 변경:
  - `data/weather.json`에 `tsuyu_normals`(긴키 매우입·매우명 평년 + 최근 7년 실적) 최상위 추가
