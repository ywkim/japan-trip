# 2026-06-01 — 6/1 점심 시간대 재구성: 사가노유 식사 + eX cafe 디저트 분리

## Status

Accepted

## Context (왜)

사용자가 eX cafe 항목을 재검토하며 다음을 지적했다: "eX cafe는 디저트만, 식사는 추가".

현재 구조:
- 10:30 사이호지 (사찰 방문, ~45분)
- 12:00 eX cafe (당고 직화구이 + 말차 빙수) — 현재 유일한 점심

문제: eX cafe는 100년 고택 카페로 시그니처가 당고 직화구이(체험)와 말차 빙수(디저트)인데, 4인 1오더 필수 정책 때문에 풀 식사(세트) 대신 **디저트/카페 경험**에 중점을 두는 것이 자연스럽다. 

사이호지 10:30→12:00 eX cafe 사이 1.5시간 간격은 사이호지 방문(~1시간) 후 충분한 식사 시간이 필요한 상황이다.

## Decision (무엇)

**6/1 점심 시간대를 두 단계로 분리한다:**

1. **사가노유(嵯峨野湯) 11:30** — 파스타·팬케이크 (식사)
   - {{arashiyama}}역 인근 전통 기와 건물의 이탈리안 파스타 전문점
   - 타베로그 3.5 평점, 예약 권장
   - 소요: 사이호지(버스 18분) → 사가노유(11:30-12:15, ~45분 식사)

2. **eX cafe 12:00** — 당고 직화구이 + 말차 빙수 (카페/디저트)
   - 타이밍: 사가노유에서 도보 5분 이내
   - 역할: 전 시간 식사 후 달콤한 당고 직화 체험 + 말차 빙수(4인 쉐어)
   - 1인 1오더 요건: 당고 세트(¥1,320) 충족 가능

3. **연결 동선:**
   - 사이호지 11:15 exit → 버스 18분 → 아라시야마역 근처 사가노유 11:35 도착
   - 11:45~12:15 식사
   - 12:15 사가노유 exit → 도보 5분 → eX cafe 12:20 도착
   - 12:30~13:00 당고·말차 빙수 디저트 (또는 12:00 arrive, 그 자리에서 기다렸다 order)

데이터: 
- `data/itinerary.json` places 레지스트리에 `saganoyu` 추가
- 6/1 items에 새 item[4] 사가노유(11:30) 삽입, 기존 eX cafe(12:00)는 items[5]로 이동
- eX cafe 메모 명확화: "당고 직화구이와 말차 빙수 전문 — 카페/디저트 시간"
- eX cafe arrive_from 업데이트: 사가노유에서 도보 5분

검사(validate):
- 검사 K (place registry): {{arashiyama}} 참조로 병기 통과
- 검사 I (food_quality): 사가노유 타베로그 출처·평점·예약 정보 기재 통과
- 215개 유닛 테스트 전부 통과

채택하지 않은 대안:
- **점심을 13:30 금각사 이후로 미루기**: 금각사(15:00 료안지, 이어서 저녁 18:30) 동선상 불편, 오후 늦은 식사는 당일 체력 저하 초래
- **eX cafe만 유지 (현상유지)**: 사용자 지시에 불부합
- **% Arabica·파인투에스프레소 등으로 대체**: 커피·빵 위주는 식사(한 끼)로 부족, 파스타가 더 적절

## Consequences (그래서)

**긍정**:
- eX cafe는 100년 고택의 **당고 직화 체험 + 말차 빙수 디저트**에 집중 — 정체성 명확화
- 점심은 사가노유 파스타로 충분한 포만감 확보 — 시부모 동반 여행의 에너지 관리
- {{arashiyama}}(嵐山) 동선 일원화 — 11:30~13:00 사이 두 가게 모두 걷거리 5분 이내
- 카페 시간(12:00) 타이밍이 자연스러움 — 점심 후 디저트 슬롯

**부정·트레이드오프**:
- 사가노유가 신규 항목이므로 일정이 1개 증가 (일정 밀도 ↑)
- 예약 권장(사가노유)이므로 사전 예약 필요 (선택사항이나 권장)

**후속 행동**:
- 사가노유 예약 가능 시 booking-checklist.json에 예약 항목 추가(선택)
- eX cafe 블로그 리뷰 Playwright 세션 위임 (기존 계획대로)
- 사가노유 Tabelog 메뉴·가격 갱신(현지 확인 또는 Playwright 후속 세션)

**영향 받은 파일**:
- `data/itinerary.json` (places 레지스트리 + 6/1 items 순서 변경)
- 본 일지
