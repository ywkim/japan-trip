# 2026-06-02 — 6/2 오후를 카덴쇼 체크인 15:00에 맞춰 재배치

## Status

Accepted

## Context (왜)

- 카덴쇼(우메코지 카덴쇼) **체크인 = 15:00**(사용자 확정). 직전 상태는 디저트 14:30~16:00 후 16:00경 입실로, **입실 가능 시각(15:00)보다 1시간 늦게** 방에 드는 동선이었다.
- 사용자 지시: "돈키호테·요도바시(봉인팩)는 유지하고, **디저트를 앞당겨 15:00 입실에 맞춰**."
- 6/2는 폭우 우천 실내 코스(`docs/decision-log/2026-06-01-6-2-rain-indoor-pivot.md`)라 야외 노출이 택시 승하차뿐 — 시각만 당기면 되고 동선·장소는 그대로.

## Decision (무엇)

- **6/2 오후 시각을 앞으로 당겨 15:00 입실에 정렬**:
  - 이세탄·포르타(+봉인팩) 13:00 → **12:30**
  - 말차 디저트 14:30 → **13:30**(권장 13:30~14:45, "15:00 입실 맞춤" 명시)
  - **15:00 "카덴쇼 복귀·입실(체크인)" 항목 신설** — 교토역→카덴쇼 택시 leg를 17:15 온천 항목에서 이 항목으로 이동.
  - 17:15 온천 항목은 카덴쇼 관내 이동이라 **arrive_from(택시 leg) 제거**(조식·가이세키와 동일하게 leg 없는 항목).
- 라멘코지 점심(11:30)·온천(17:15)·가이세키(18:00) 시각은 유지. 택시 총 3회 불변(시오→카덴쇼·카덴쇼→교토역·교토역→카덴쇼).
- **검토 후 기각**: 디저트 유지·16:00 입실(사용자가 15:00 입실 명시), 온천까지 당기기(시부모 방 휴식 시간 확보 위해 17:15 유지).

## Consequences (그래서)

- 긍정: 입실 가능 즉시 방 확보(15:00), 입실~온천(17:15) 2시간 방 휴식으로 시부모 충분한 쉼, 봉인팩 조달 동선 유지.
- 부정·트레이드오프: 점심~디저트 간격이 약 1시간으로 다소 빠듯(라멘 후 바로 쇼핑·디저트). 디저트 피크 전이라 오히려 대기 짧을 수 있음.
