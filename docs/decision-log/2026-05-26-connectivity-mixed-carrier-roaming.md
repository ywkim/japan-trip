# 2026-05-26 — 데이터/연결성: 회선별 분리 (부부 SKT baro 가족로밍 + 시부모 KT 데이터함께ON)

## Status

Proposed (옵션·1인당 GB 비교 완료, 최종 선택 사용자 확인 대기)

## Context (왜)

- `data/booking-checklist.json`의 `esim` 항목(마감 2026-05-28)은 2026-05-24 리서치에서 **4인 일괄 Klook eSIM(~₩11K/인)**을 권장했다.
- 2026-05-26 사용자가 SKT **T로밍 baro 요금제 + Club T 로밍** 혜택 화면(공항버스·푸드코트·귀국 후 디저트 할인)을 공유, 로밍을 대안으로 검토 요청("먼저 비교하고 합리적으로 결정").
- 확인 결과 **4인의 통신사가 갈린다**: 부부(영욱·소연) SKT, 시부모 KT. baro 가족로밍은 **전원 SKT**여야 성립하므로 단일 일괄안이 성립하지 않는다.
- 시부모는 고령으로 **폰 eSIM 설치·이중유심 전환·기기 미지원** 리스크가 부담 → 번호 유지·무설치가 안전·이동부담 측면에서 중요(레포의 시부모 동반 운영 원칙과 부합).
- 실시간 재검색(2026-05-26): SKT baro 6GB ₩39,000(프로모션 →8GB)·가족로밍 +₩3,000(최대 5명), KT 데이터함께ON ₩33,000·아시아 4GB·**같은 KT 2명 공유**·통화 119원/분, KT 중국/일본 알뜰 ₩25,000/5일/3GB(단일), Klook eSIM ~₩11,000/인. 하루종일 무제한형(₩15,000~19,000/일)은 4일 환산 고가로 제외.

## Decision (무엇)

- **회선별 옵션을 1인당 GB까지 비교하고, 최종 선택은 사용자 확인에 둔다.** 통신사 갈림(부부 SKT·시부모 KT)으로 baro 가족로밍은 부부만 가능.

| 구성안 (4일, 공유 균등환산) | 영욱 | 소연 | 시부 | 시모 | 합계 | 비용 |
|---|---|---|---|---|---|---|
| ① 전원 Klook eSIM (각 독립) | 4GB | 4GB | 4GB | 4GB | 16GB | ₩44,000 |
| ② 부부 baro 8GB 공유 + 시부모 KT 데이터함께ON 4GB 공유 | ≈4GB | ≈4GB | ≈2GB | ≈2GB | 12GB | ₩75,000 |
| ③ 부부 eSIM + 시부모 KT 데이터함께ON | 4GB | 4GB | ≈2GB | ≈2GB | 12GB | ₩55,000 |
| ④ 부부 baro 16GB 공유 + 시부모 KT 데이터함께ON | ≈8GB | ≈8GB | ≈2GB | ≈2GB | 20GB | ₩95,000 |

- 트레이드오프: **데이터 양·가성비 우선 → ①**(4GB/인을 최저가로 제공, 단 시부모 eSIM 설치 필요). **시부모 무설치·전원 번호 유지·baro 통화·Club T 혜택 우선 → ②**. 로밍안(②④)은 비용을 더 써도 데이터가 늘지 않고 시부모는 2GB/인으로 오히려 적다.
- 제외: SKT/KT 하루종일 무제한형(4일 환산 ₩60,000~76,000/인 고가).
- 본 항목은 **비교·미구매**. status `미정` 유지, 선택·결제·개통 완료 시 `확정`(`confirmed_booking`) 승격 + 별도 일지.

## Consequences (그래서)

- 긍정: 시부모 무설치·번호 유지로 이동 중 분실·미지원 리스크 제거. 부부는 결제·QR·OTP 회선(SKT) 유지 + Club T 공항/식음 혜택으로 baro 프리미엄(eSIM 대비 ~₩20K) 상당 부분 상쇄. baro 통화로 그룹 내 통화 무료.
- 부정·트레이드오프: 일괄 eSIM(₩44K) 대비 합계 ~₩31K 증가. 통신사 2곳 가입·설정으로 출국 전 준비 절차가 둘로 나뉨.
- 후속 행동: 사용자가 ①~④ 중 선택(데이터·가성비 vs 시부모 편의). 선택 후 출국 전 해당 가입·설정(②④는 부부 baro 가족로밍 + 시부모 KT 데이터함께ON, ①은 4인 eSIM, ③은 부부 eSIM + 시부모 KT). 개통·결제 완료 시 체크리스트 승격 + 일지. (프로모션 2026-04-23~08-21 데이터 증정 적용 여부, "첫 로밍 4배" 자격은 가입 시 확인.)
