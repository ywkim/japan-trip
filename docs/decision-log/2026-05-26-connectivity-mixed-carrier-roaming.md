# 2026-05-26 — 데이터/연결성: 실사용 right-sizing (영욱 1회선 + 나머지 핫스팟)

## Status

Proposed (옵션·1인당 GB 비교 완료, 최종 선택 사용자 확인 대기)

## Context (왜)

- `data/booking-checklist.json`의 `esim` 항목(마감 2026-05-28)은 2026-05-24 리서치에서 **4인 일괄 Klook eSIM(~₩11K/인)**을 권장했다.
- 2026-05-26 사용자가 SKT **T로밍 baro 요금제 + Club T 로밍** 혜택 화면(공항버스·푸드코트·귀국 후 디저트 할인)을 공유, 로밍을 대안으로 검토 요청("먼저 비교하고 합리적으로 결정").
- 확인 결과 **4인의 통신사가 갈린다**: 부부(영욱·소연) SKT, 시부모 KT. baro 가족로밍은 **전원 SKT**여야 성립하므로 단일 일괄안이 성립하지 않는다.
- 시부모는 고령으로 **폰 eSIM 설치·이중유심 전환·기기 미지원** 리스크가 부담 → 번호 유지·무설치가 안전·이동부담 측면에서 중요(레포의 시부모 동반 운영 원칙과 부합).
- 실시간 재검색(2026-05-26): SKT baro 6GB ₩39,000(프로모션 →8GB)·가족로밍 +₩3,000(최대 5명), KT 데이터함께ON ₩33,000·아시아 4GB·**같은 KT 2명 공유**·통화 119원/분, KT 중국/일본 알뜰 ₩25,000/5일/3GB(단일), Klook eSIM ~₩11,000/인. 하루종일 무제한형(₩15,000~19,000/일)은 4일 환산 고가로 제외.
- **사용자 제시 실사용 추정(2026-05-26)**: 영욱 ~3.5GB·소연 ~450MB·시부 ~55MB·시모 ~9MB = 합계 ≈ 4GB(영욱 ~88%). 시부모는 데이터를 거의 쓰지 않음 → 4인 각자 플랜은 과대.

## Decision (무엇)

- **실사용(~4GB·영욱 집중)에 맞춰 right-size: 영욱 1회선 데이터 + 나머지 핫스팟.** 통신사 갈림(부부 SKT·시부모 KT)으로 baro 가족로밍은 부부만 가능하나, 시부모 실사용 64MB는 유료 가입이 불필요.
  - 영욱 라인(본인 3.5GB + 핫스팟 ≈ 4GB): **(a) baro 6GB(프로모션 8GB) ₩39,000**(SKT 번호 native·baro 통화·Club T 혜택) 또는 **(b) Klook eSIM 4GB ~₩11,000**(최저가·데이터만, SKT 통화/SMS 유지). 택1은 사용자 확인.
  - 소연(450MB): 핫스팟 ₩0, 자기 라인 원하면 baro 결합 +₩3,000 또는 eSIM ~₩11,000.
  - 시부모(KT, 64MB): **데이터 유료 가입 불필요·핫스팟 ₩0.** 수신전화 필요 시 KT 음성 로밍만 확인.
  - **합계 ₩11,000~39,000.**
- 미채택(과대 — 실사용 ~4GB 대비 12~20GB 제공): ①전원 eSIM ₩44,000 · ②부부 baro+시부모 KT함께ON ₩75,000 · ③부부 eSIM+시부모 KT ₩55,000 · ④부부 baro 16GB+시부모 KT ₩95,000. 1인당 GB 비교표는 `docs/booking-research-2026-05-24.md` §3 참고.
- 제외: SKT/KT 하루종일 무제한형(4일 환산 ₩60,000~76,000/인 고가).
- 본 항목은 **비교·미구매**. status `미정` 유지, 택1·결제·개통 완료 시 `확정`(`confirmed_booking`) 승격 + 별도 일지.

## Consequences (그래서)

- 긍정: 실사용에 맞춰 ₩11,000~39,000으로 right-size(이전 ₩44~95K 대비 절감). 시부모 무가입·무설치(거의 미사용), 영욱 baro 선택 시 SKT 번호·baro 통화·Club T 공항 혜택 유지.
- 부정·트레이드오프: 소연·시부모가 영욱 핫스팟에 의존(분리 시 데이터 없음) — 단 시부모 실사용 64MB로 위험 작음. 영욱 eSIM 택 시 SKT 데이터 로밍 차단·이중유심 설정 필요.
- 후속 행동: 영욱 라인 baro(₩39K) vs eSIM(₩11K) 택1 확인 → 출국 전 가입·설정, 도착 후 데이터 ON+핫스팟. 소연 자기 라인 여부 결정. 개통·결제 완료 시 체크리스트 승격 + 일지. (baro 선택 시 프로모션 2026-04-23~08-21 데이터 증정·"첫 로밍 4배" 자격 가입 시 확인.)
- 영향 받은 파일: `data/booking-checklist.json`(`esim` 항목), `docs/booking-research-2026-05-24.md`(§3 및 요약 표), 빌드 산출물(`viz/checklist.html` 등 `scripts/build_index.py` 재생성).
