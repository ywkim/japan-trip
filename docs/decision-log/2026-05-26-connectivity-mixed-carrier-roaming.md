# 2026-05-26 — 데이터/연결성: 회선별 분리 (부부 SKT baro 가족로밍 + 시부모 KT 데이터함께ON)

## Status

Accepted

## Context (왜)

- `data/booking-checklist.json`의 `esim` 항목(마감 2026-05-28)은 2026-05-24 리서치에서 **4인 일괄 Klook eSIM(~₩11K/인)**을 권장했다.
- 2026-05-26 사용자가 SKT **T로밍 baro 요금제 + Club T 로밍** 혜택 화면(공항버스·푸드코트·귀국 후 디저트 할인)을 공유, 로밍을 대안으로 검토 요청("먼저 비교하고 합리적으로 결정").
- 확인 결과 **4인의 통신사가 갈린다**: 부부(영욱·소연) SKT, 시부모 KT. baro 가족로밍은 **전원 SKT**여야 성립하므로 단일 일괄안이 성립하지 않는다.
- 시부모는 고령으로 **폰 eSIM 설치·이중유심 전환·기기 미지원** 리스크가 부담 → 번호 유지·무설치가 안전·이동부담 측면에서 중요(레포의 시부모 동반 운영 원칙과 부합).
- 실시간 재검색(2026-05-26): SKT baro 6GB ₩39,000(프로모션 →8GB)·가족로밍 +₩3,000(최대 5명), KT 데이터함께ON ₩33,000·아시아 4GB·**같은 KT 2명 공유**·통화 119원/분, KT 중국/일본 알뜰 ₩25,000/5일/3GB(단일), Klook eSIM ~₩11,000/인. 하루종일 무제한형(₩15,000~19,000/일)은 4일 환산 고가로 제외.

## Decision (무엇)

- **회선별 최적안으로 분리한다.**
  - 시부모 2인(KT): **KT 데이터함께ON 1개(₩33,000) 공유** — 번호 유지·eSIM 설치 불필요·4GB 공유.
  - 부부 2인(SKT): **baro 가족로밍 6GB(프로모션 8GB) ₩42,000**(영욱 대표+소연 결합 +₩3,000) — 번호 유지·baro 통화 무료·Club T 로밍 혜택.
  - 4인 합계 ≈ **₩75,000**. 가입만으로는 과금 없음(해외 실사용 시 과금).
- 채택하지 않은 대안:
  - 4인 일괄 Klook eSIM(~₩44,000): 최저가지만 시부모 eSIM 설치 부담·번호 비활성으로 고령 동반 여행에 비권장.
  - 부부 Klook eSIM 2개 + 시부모 KT함께ON(~₩55,000): ₩20K 저렴하나 부부 SKT 번호 비활성·설치 필요. **비용 우선 시 대안으로 보존**.
  - SKT/KT 하루종일 무제한형: 4일 환산 ₩60,000~76,000/인으로 고가.
- 본 항목은 **권장·미구매**다. status는 `미정` 유지, 결제·개통 완료 시 `확정`(`confirmed_booking`) 승격 + 별도 일지.

## Consequences (그래서)

- 긍정: 시부모 무설치·번호 유지로 이동 중 분실·미지원 리스크 제거. 부부는 결제·QR·OTP 회선(SKT) 유지 + Club T 공항/식음 혜택으로 baro 프리미엄(eSIM 대비 ~₩20K) 상당 부분 상쇄. baro 통화로 그룹 내 통화 무료.
