# 2026-06-01 — 6/1 점심 최종 선택: 권타로 금각사점 (요시무라 → 대체)

## Status

Accepted

## Context (왜)

6/1 점심 후보로 **음식·경치** 우선 기준에 따라 아라시야마 요시무라(嵐山よしむら, Tabelog 3.53·도게쓰교 창가 파노라마 뷰)를 선택했다. 다만 현장 예약 불가 정책으로 **대기 시간 약 1시간 20분** 예상.

6월의 교토는 **기온 30도대·습도 높음(장마 전 무더위)**, 동반자는 **시부모 2인(고령자)**. 1시간 이상 대기는:
- 실내 대기 공간 협소 → 고령자 체력 부담
- 근처 보호처 없음 → 햇볕 노출 위험  
- 다음 일정(13:30 금각사)과 뒤이은 료안지(15:30)까지 연계 시간이 타이트 → 지각 리스크

**실행 가능성 vs 체험 우선도의 재정렬**: 사진·경치도 중요하지만 "4인 가족 편안한 여행"이 1차 목표.

## Decision (무엇)

**아라시야마 요시무라를 권타로 금각사점으로 대체한다.**

- **권타로 금각사점** (京都府京都市右京区金閣寺町)
  - Tabelog 3.45 (1910 리뷰) — 역사 1910년 창업, 교마치야 건물·정원
  - 예약 권장 (점심 12:00·13:30 주로 가능) → 대기 최소(5~10분 예상)
  - 메뉴: 황금의 일즙(일본 국수 육수) 조림국수·소바·세트 (유명 소재지 금각사 남측 200m)
  - 정원 카테고리 1인 ¥2,500~3,500 (요시무라보다 약간 상향)
  
**채택하지 않은 대안**:
- **요시무라 유지** — 1시간 대기, 시부모 체력 고려 불가능
- **다른 카페·식사 장소로 변경** — 금각사 접근성 고려하면 권타로가 최선

## Consequences (그래서)

- 예약으로 점심 12:00~13:10 확정 → 13:30 금각사 충분히 도착(금각사 남측 200m, 도보 3분)
- 고령자 친화: 예약→현장 진입→식사→퇴장 스트레스 극소화
- 교마치야 정원 비경(실내·조명 조절 가능) + 여름 무더위 피할 수 있음
