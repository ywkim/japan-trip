# 2026-05-26 — 사이호지(苔寺) 참배 예약 가능성 확인

## Status

Accepted

## Context (왜)

- #44에서 사이호지(西芳寺·고케데라) 방문·예약 가능성 확인이 후속 과제로 남았다. 사이호지는 사전 예약제(예약 없이 입장 불가)로, 시부모(60~70대) 동반 시 사경(写経) 의무·접근성이 변수다.
- 방문 후보일은 6/1(월)·6/2(화) 오전(교토 서북부 동선). 실제 예약 가능 여부·방법·리드타임을 시점 정보로 확인할 필요가 있었다.

## Decision (무엇)

- 공식 사이트(intosaihoji.com) 기준 예약 방법·리드타임·참배료·사경 면제·접근성·6월 이끼 상태를 실시간 리서치하여 `docs/saihoji-reservation-2026-06.md`에 기록했다.
- **본 작업은 "가능성 확인"까지** — 실제 계정 등록·예약·일정 편입은 사용자 결정 사항(시간 민감: 방문 7일 전 0시 선착순 개방).
- 핵심: 온라인 예약 가능(계정당 2명 → 4인 = 계정 2개), 1인 ¥4,000+¥110, 사경 면제 상담 가능·의자석 있음, **휠체어 불가**, 6월=이끼 최성수기.

## Consequences (그래서)

- 긍정: 예약 가능 여부·절차·시부모 적합성이 출처와 함께 정리됨. 6/1·6/2 방문이 시점상 신청 가능 구간임을 확인.
- 부정·트레이드오프: 선착순이라 잔여 슬롯은 7일 전 개방 후 직접 확인 필요. 휠체어·거동 매우 불편자는 부적합. 교토역에서 버스 60분으로 동선 조정 필요.
- 후속: ① 방문 의향 확정 시 즉시 계정 2개 등록·신청 ② 6/1·6/2 기존 오전 일정과 충돌 조정 ③ 결정 시 `data/itinerary.json`·`booking-checklist.json` 반영(별도).
- 영향 파일: 신규 `docs/saihoji-reservation-2026-06.md`, `README.md`, `CLAUDE.md`.
