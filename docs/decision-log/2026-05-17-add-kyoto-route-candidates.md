# 교토 일정 후보 코스 3개 추가

- **날짜**: 2026-05-17
- **주제**: 확정 동선(빡빡형) 외 대안 코스 3개 추가
- **산출물**: `data/itinerary.json` → `route_candidates` 배열, `viz/itinerary.html` 후보 코스 섹션

## 배경

교토 5/31~6/3, 숙소(에어비앤비 시오 2박 + 우메코지 카덴쇼 1박), 항공은 확정 상태. 확정 코스(빡빡형)를 기준으로 동선만 다른 대안을 비교·검토하기 위해 추가.

## 추가 후보 코스

### 여유형 (총 도보 약 9km)
- 주제: 시부모 부담 최소화, 하루 2~3 방문지, 카페 휴식 포함
- 빡빡형(13km) 대비 도보 31% 감소
- 핵심 관찰: 후시미이나리는 입구 1구간만, 아라시야마는 오후 자유시간 추가

### 서북 사찰 집중형 (총 도보 약 14km)
- 주제: 금각사·료안지·닌나지·묘신지 서북 라인 + 우지 뵤도인(세계유산)
- 빡빡형과 첫날 동일, 2일차에 묘신지(대규모 선종 사찰) 추가
- 핵심 관찰: 우지는 JR 나라선으로 교토 당일치기 가능 (30분). 뵤도인은 입장권 사전 구매 권장

### 미식+문화 체험형 (총 도보 약 12km)
- 주제: 니시키 시장·교토 전통 요리·도예 체험·폰토초 골목 중심
- 핵심 관찰: 도예 공방 체험은 사가 도예 기념관에서 가능. 교토 철도박물관은 시부모 흥미 시 선택

## 데이터 구조 변경

- `data/itinerary.json`에 `route_candidates` 배열 신설
- 기존 `days` (확정 코스)는 변경 없음 — 후보는 별도 배열로 격리
- `scripts/build_index.py`의 `build_itinerary()` 함수: 후보 코스를 `<details>/<summary>` 접이식으로 렌더
- `tests/test_build_index.py`: `test_route_candidates_rendered_in_itinerary` 테스트 추가

## 합의·보류

- 합의: 후보 코스는 참고용. 실제 동선은 확정 코스(`days`) 기준으로 출발 전 최종 결정
- 보류: 후보 코스 중 혼합(예: 빡빡형 2일 + 여유형 1일) 여부는 미결
- 다음 단계: 출발 전 최종 코스 1개 선택 또는 혼합 확정 → `days` 업데이트
