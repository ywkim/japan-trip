# 2026-05-30 — 첫날(5/31) 저녁 코스에 야사카 신사 야경 + 가모강 산책 추가

## Status

Accepted

## Context (왜)

- 사용자 요청: "야사카 신사가 밤에 이쁘다 — 거기 앞 거리에서 숙소(에어비앤비) 짐 풀고 저녁 먹고 신사 보고 가모강 따라 걷는 코스를 첫째날에 리서치해서 추가."
- 사실관계 확인: 확정 1·2박 숙소 **시오(Shio) 마치야는 중교구 니조역 도보 7분**으로 야사카 신사(기온 동쪽)와 약 3~4km 떨어져 있다. "야사카 앞 거리에서 짐을 푼다"는 그림과 실제 숙소 위치가 다르다. → `AskUserQuestion`으로 확인, 사용자가 **"시오에 짐 풀고 기온 원정"** 선택.
- 기존 첫날(5/31)은 오후 도착(KIX 15:15) 피로를 고려해 시오 인근 저녁(中華 大鵬)만 두고 동산 관광은 6/2로 미뤄둔 상태였다.
- 실시간 리서치(2026-05-30, WebSearch):
  - 야사카 신사는 **24시간 개방·무료**, 저녁이면 시조도리 동쪽 끝 **서루문(西楼門)**이 주홍빛으로 점등되고 본전 맞은편 **무전(舞殿)**에 수백 개 등롱이 켜져 ~21:30경까지 야경. 부적·고슈인은 16:00 마감. (japan-guide, JNTO, magical-trip 등)
  - **가모강 납량상(納涼床·川床)** 2026년 운영기간 5/1~(야간 床 5/9~9월말) → 5/31 강가에 床이 늘어선 여름 야경. 시조대교~폰토초~산조대교 강변길은 평탄해 고령자 산책 적합. (교토시 공식 관광 Navi, kyoto-yuka.com)
  - 식당: 사용자가 떠올린 "권태로"는 **祇園店이 없음**(본점은 가모강 서편 麩屋町通四条·우동스키 나베·¥5,000대·~21:30·수휴). 야사카 앞 거리에서 검증된 대안으로 **手打ちうどん おかる**(게이한 기온시조역 도보 5분·시조도리 남측 기온·1925년 창업·미쉐린 빕구르망 단골·17:00~심야·일요일 영업·명물 ねぎうどん) 채택. (Tabelog 八坂神社 주변 우동, Retty, 미쉐린)
  - 교통: 니조역 → 지하철 도자이선 二条→三条京阪 + 게이한 三条→祇園四条. 귀로 三条京阪→二条 도자이선 1회. (교토시 교통국 도자이선, jorudan)

## Decision (무엇)

- `data/itinerary.json` 첫날(5/31)의 "시오 인근 저녁(大鵬)" 단일 항목을 **기온 야경 원정 4항목**으로 교체한다:
  1. 18:45 야사카 앞 저녁 — **手打ちうどん おかる**(우동 노포)
  2. 19:45 **야사카 신사 야경**(서루문·무전 등롱)
  3. 20:15 **가모강 산책**(시조대교→폰토초→산조대교)
  4. 21:00 산조 → 시오 복귀(취침)
- 17:45 체크인 항목 메모를 "무거운 짐은 시오에 먼저 풀고 가벼운 차림으로 기온 원정"으로 갱신.
- 大鵬(쓰촨·타베로그 3.76)은 삭제하지 않고 おかる 항목의 **도착편 지연 시 대안**(기온 원정 생략)으로, 좌석 여유·個室이 필요하면 **権太呂 四条본점**을 보조 대안으로 메모에 보존.
- 채택하지 않은 대안:
  - "기온에 먼저 들렀다 시오 늦은 체크인" — 무거운 짐을 들고 이동하는 부담·코인락커 필요로 기각(사용자 선택과도 불일치).
  - 식당 "권태로 祇園店" — 실제 존재하지 않음(리서치로 확인). 본점은 동선상 가모강 서편이라 야사카 앞이 아님.
- 일자 메타 갱신: `walking_km` 1→3, `day_label`, `walking_km_source`, `pass_recommendation`, 트립 `walking_km_total` 13→15, `transit_pass_playbook`의 5/31 항목을 기온 야경 동선으로 정정.

## Consequences (그래서)

- 긍정: 첫날 저녁이 단순 인근 식사에서 "야사카 야경 + 가모강 산책"이라는 사용자가 원한 미감·사진 동선으로 강화. 야사카는 24시간·무료라 별도 예약·입장료 부담 없음. おかる는 17:00~심야 영업이라 오후 도착·지연에도 유연.
