# 2026-06-01 — 6/1 저녁 만자라테이 → 御料理めなみ 완전 교체

## Status

Accepted

(`docs/decision-log/2026-05-30-dining-reservations.md`의 6/1 저녁 만자라테이 선정을 본 결정이 대체. 만자라테이 번역 페이지 추가 일지도 본 결정으로 Superseded)

## Context (왜)

PR #88에서 6/1 저녁을 **만자라테이(まんざら亭 先斗町店)**로 잡고 번역 페이지(`docs/manzaratei-review-translation.md` → `viz/manzaratei-review.html`)까지 추가했으나, 운영 검토에서 동반자 조건과 어긋남이 드러났다.

- **동반자**: 부부 + 시부모 4인. 건강상 **술을 안 마심**. 아버님은 **두부를 싫어함**.
- **만자라테이 문제**: ① 飲み放題(무제한 음주)를 전제한 **이자카야 업태** → "술 없이 편한 한 끼" 성격과 어긋남 ② 1인 ¥6,000~8,000으로 **예산(1인 ¥4,000 이하) 초과** ③ 유바(두부피)·창작요리 중심.
- **추가 제약(끼니 중복)**: 5/31 저녁이 카네요(장어), 6/1 점심이 권타로(소바·우동스키). 저녁은 **장어·소바·우동·면류·튀김과 겹치지 않아야** 함.

대안을 일본어 1차 출처로 검증한 결과 대부분 탈락:
- 綴·六角や — Tabelog 분류가 **居酒屋**(술 곁들임 전제)
- 順正 — **유두부 전문**(두부 회피와 정면 충돌)
- 有喜屋 先斗町本店 — 소바 노포로 조건은 좋으나 **정기휴일이 월요일**(6/1 휴무) + 점심 소바와 면류 중복
- 京味菜 わたつね — 가성비 좋으나 **소바·카츠동·굴튀김이 간판** → 점심 소바·튀김과 콘셉트 중복

## Decision (무엇)

**6/1 저녁을 御料理めなみ(메나미)로 완전 교체하고, 만자라테이 번역 페이지를 메나미 번역 페이지로 대체한다.**

메나미가 모든 제약을 통과:
- **업태**: 1939년 창업 오반자이 노포, 스스로 **「お食事処」**(이자카야 아님) → 술 없이 밥·오반자이·생선구이로 한 상 완결
- **예산**: 공식 메뉴 가격으로 4인이 술 없이 푸짐하게(오반자이 모둠 2·계란말이·생선구이·오리구이·밥·국·호지차) 시켜 **¥15,730 = 1인 약 ¥3,930**. Tabelog 평균 ¥8,000은 사케·회모둠·소꼬리찜 등 술·고가항목 포함분
- **두부 비중심**: 두부 메뉴(京とうふ小鍋·生ゆば)만 빼면 야채·생선·고기 중심
- **6/1(월) 영업**: 정기휴일 **수요일**
- **중복 회피**: 장어(5/31)·소바/우동/튀김(6/1 점심)과 안 겹침
- **Tabelog 3.52 / 331**, 교토시야쿠쇼마에역 1번출구 도보 2분(기존 만자라테이 동선 거의 동일, 더 가까움)

**채택하지 않은 대안**(위 Context의 탈락 사유와 동일): 만자라테이(이자카야·예산·두부) / 綴·六角や(이자카야) / 順正(두부 전문) / 有喜屋(월요일 휴무·면류 중복) / わたつね(소바·튀김 콘셉트 중복).

**출처 정책 준수**: 식당 사실(메뉴·가격·영업·휴무·평점)은 일본어 1차 출처(공식 menami.jp·Tabelog ID 26001266)에서만 인용. 한국어 블로그 사실 인용 없음.
