# 2026-05-18 — 아라시야마 두부 가이세키 가게 확정: 쇼라이안 (松籟庵)

## 트리거

PR #29 후속 세션이 itinerary의 "시그라쿠 토후"를 Google Maps에서 검색 못 해 텐류지에서 가장 가까운 두부 가이세키 매물(쇼라이안, 도보 7분 500m)로 추정해 측정했다. `docs/decision-log/2026-05-18-transit-mcp-measurements.md`에서 "예약 시 사용자 확인 필요"로 보류.

본 세션에서 사용자가 직접 한국 블로그·일본 음식점 사이트(Tabelog·Klook·waug·kkday·shiroang)를 교차 확인해 후보 3종을 비교 → **쇼라이안 (松籟庵)으로 확정**.

## 핵심 관찰

- **"시그라쿠"의 원천 추적 불가**: 한국어 음역으로 통용되는 가게가 존재하지 않음. 구버전 `docs/kyoto-itinerary-may-2026.md`(5/24~27 폐기 버전)에도 동일 표기 — 첫 itinerary 작성 시 사용자 메모/스크린샷의 음역 오류로 추정.
- **leg 4(텐류지→쇼라이안) 측정값 유지**: 후속 세션이 측정한 도보 7분 500m·이동 메타·Google Maps Directions URL은 정확하므로 변경 불필요.
- **예약 우선순위 상승**: 쇼라이안 미슐랭 1스타·아라시야마 점심 시간 혼잡으로 사전 예약 필수. `docs/kyoto-itinerary-may31-jun3-2026.md` §3 보류 항목에 강조 보강.
- **decision-log 불변 원칙 준수**: 직전 일지(2026-05-18-transit-mcp-measurements.md)의 "예약 시 사용자 확인 필요" 메모를 편집하지 않고 본 새 일지로 승계·확정.
