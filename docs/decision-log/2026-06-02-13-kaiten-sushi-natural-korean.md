# 2026-06-02 — 회전초밥 비교 페이지 번역 자연스러운 한국어로 다듬기

## Status

Accepted

## Context (왜)

- 사용자 지적: **"자연스러운 한국어가 아닌데."** 쿠라스시 전체 메뉴 번역(`2026-06-02-12`)을 비롯해 페이지 곳곳에 일본어 음차·직역투(시메사바·본마구로·우니·게소·네타·샤리·철화김초밥·아세롤라 부리히라 등)가 그대로 남아 가독성을 해침.

## Decision (무엇)

- **`docs/kyoto-station-kaiten-sushi-translation.md`의 세 점포 메뉴·설명을 자연스러운 한국어로 통일** — 음차어를 한국어 어식 명칭으로 교체:
  - 시메사바→고등어 초절임, 본마구로→참다랑어, 우니→성게알, 게소→오징어 다리, 철화김초밥→참치 김초밥, 마구타쿠→참치 단무지 군함, 빈초마구로→날개다랑어, 네기마구로→파참치, 하마치→방어, 엔가와→광어 지느러미살, 츠부가이→골뱅이, 카이센동→해산물 덮밥, 焼きはらす→연어 뱃살구이, 구운 옥수수→옥수수 구이 등.
  - "네타·샤리" 같은 업계 일본어 → "재료·밥알". 초밥 단위 "관" → "점"으로 통일.
- **표 헤더·후기 인용문도 동일 기조로 윤문** — 의미는 보존, 출처·평점 신호(★·타베로그 점수)는 유지.
- 가격·출처·이미지·구조는 변경 없음(번역 표현만 수정).

## Consequences (그래서)

- 긍정: 가족(시부모 포함)이 읽기에 매끄러운 한국어. 세 점포 메뉴 용어가 일관.
- 부정·트레이드오프: 일부 일본 고유 명칭(바테라·사쿠라·빈토로 등)은 대체어가 없어 음차+괄호 설명 유지.
- 영향 받은 파일: `docs/kyoto-station-kaiten-sushi-translation.md`(번역 표현 윤문).
