# 2026-06-02 — 쿠라스시 섹션 전체 메뉴 번역 확장

## Status

Accepted

## Context (왜)

- 직전 PR(`2026-06-02-11`)에서 비교 페이지에 ③ くら寿司를 추가했으나, 메뉴는 5행 요약 표만 있었음(①② 大起水産·寿しのむさし는 단계별 가격 표를 갖춤).
- 사용자 지시: **"쿠라스시 메뉴 번역도 추가해."** — ①②와 같은 수준의 메뉴 번역을 요구.

## Decision (무엇)

- **`docs/kyoto-station-kaiten-sushi-translation.md` ③ 섹션 메뉴 표를 카테고리별 전체 번역으로 확장** — 공식 메뉴(`kurasushi.co.jp/menu`) 실시간 조회 기반. 니기리(¥115~300 단계)·일관 고급 네타·군함/김초밥·사이드/면류·디저트/음료로 분류, 한국어 품목명+가격(세금 포함).
- **콜라보 한정 메뉴(현재 주술회전)는 상시 메뉴와 분리** 명시 — 시기·가격 변동 주의 1줄.
- 기존 이미지 3장(네기마구로·빈토로·하마치)·후기·주의는 유지.

## Consequences (그래서)

- 긍정: 세 점포 모두 동일한 메뉴 번역 깊이로 일관. 가족이 ¥115 균일가의 실제 구성을 현장 전에 파악 가능.
- 부정·트레이드오프: 표가 길어져 유지 대상 증가. 가격은 표준 점포(西大路七条) 기준이며 도심 상위 티어 점포는 다를 수 있음(상시 메뉴 한정 주석으로 보완). 콜라보 메뉴는 회전이 빨라 추적 안 함.
- 영향 받은 파일: `docs/kyoto-station-kaiten-sushi-translation.md`(③ 메뉴 번역 표 확장).
