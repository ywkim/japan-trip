# 2026-05-11 — seasonality 점수 일괄 입력 (2026-05 시기 고정)

- 합의 (소연): **시기를 잠정적으로 2026-05 (18일 이후)로 고정**하고, 각 후보의 `data/weather.json` 2026-05 `comfort_score`를 `decision.json` `seasonality`에 그대로 입력
- 채택 스키마: **옵션 1 (단일 시기 채택)** — 단순성·즉시 비교 가능. 옵션 3(스키마 확장)은 다른 기준에도 시기 의존성이 있을 때 별도 PR로 재검토
- 입력값:
  | 후보 | 2026-05 comfort_score → seasonality |
  |---|---|
  | 오사카 | 9 |
  | 교토 | 9 (기입력) |
  | 고베 | 9 (기존 5에서 갱신 — 5에 해당하던 가정은 6월 시나리오였음) |
  | 도쿄 | 8 |
  | 삿포로 | 8 |
  | 후쿠오카 | 8 |
  | 오키나와 | 5 (5월 중순부터 우기 진입) |
- 산출물:
  - `data/decision.json` — 7개 후보 `seasonality` 입력 + 각 `notes`에 weather.json 인용
  - `viz/dashboard.html` — 인라인 데이터 동기화
- `python scripts/score.py` 출력 (현 시점, **seasonality 외 점수가 채워진 후보는 교토뿐** — 다른 후보는 seasonality 단일 점수의 정규화 결과이므로 종합 비교 아님):
  - 교토 7.60 (적용 가중치 1.00, 전 기준 입력)
  - 오사카·고베 9.00 / 도쿄·삿포로·후쿠오카 8.00 / 오키나와 5.00 — 모두 적용 가중치 0.10 (seasonality만)
- 핵심 관찰:
  - **5월은 본토 절정기** — 오사카·교토·고베 9점이 동률 최고. 후쿠오카·도쿄·삿포로 8점으로 차이 작음
  - **오키나와 5점은 후보 중 유일한 큰 페널티** — 18일 이후 우기 진입이 결정타
  - **고베 seasonality 점수는 6월(5) → 5월(9)로 큰 차이** — 시기 결정이 고베 후보의 매력도에 결정적
  - 옵션 3(스키마 확장)을 미루지만, 추후 cost·experience 등에도 시기 의존성이 확인되면 재논의 필요
