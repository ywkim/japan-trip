# 2026-05-31 — PASMO 앱 무기명 ¥0: 카드 없이 모바일 IC 운용 경로 추가

## Status

Accepted

## Context (왜)

- `docs/icoca-iphone-setup.md`는 모바일 IC카드 셋업을 ICOCA(경로 A) + 종이 카드(경로 B)
  두 갈래로만 안내해 왔다. 두 경로 모두 §1.1·§1.3에서 **Wallet에 신용/체크카드(Mastercard·Amex)
  1장 등록**을 전제한다.
- 직전 세션 질문("현대카드 없으면 PASMO?")을 계기로 2026-05-31 실시간 리서치(WebSearch/WebFetch)
  결과, 카드 종류(ICOCA/PASMO/Suica)를 바꿔도 **Apple Pay 충전의 카드 브랜드 요건은 동일**함을
  확인했다. 다만 **발급 경로**에 결정적 차이가 있었다:
  - ICOCA·Suica는 Wallet `+`에서 직접 추가되지만 **발급 시 Mastercard/Amex로 초기 충전 결제 필요**
  - PASMO는 Wallet 직접 추가가 안 되고 **PASMO 앱**을 거쳐야 하는 대신, **잔액 ¥0 무기명**으로
    카드 없이 발급 가능 → 이후 현금만으로 충전 운용
- 즉 Wallet에 등록할 적합 카드가 전혀 없는 인원(시부모 가능성)에게는 PASMO 앱 무기명 경로가
  종이 카드(KIX 현장 발권) 외 유일한 출국 전 모바일 IC 대안이다.
- 무기명 PASMO 발매는 2023-06-08 중단됐다가 **2025-03-01 재개** — 2026-05 여행 시점에는 정상 발급.
- 출처: AtaDistance(2026-05-31), Get Around Japan(2026-05-31), PASMO 나무위키(무기명 재개일).

## Decision (무엇)

- `docs/icoca-iphone-setup.md`에 **§3.4 경로 C — PASMO 앱 무기명 ¥0(카드 없이 모바일 IC 운용)**
  절을 추가한다(기존 §3.4 환불은 §3.5로 이동). §1.3 카드 호환표 아래에 "적합 카드가 없으면
  경로 C" 주의문, §5 체크리스트에 "(카드 미등록자) PASMO 앱 무기명 발급" 항목, §출처에
  클릭 가능한 근거 링크 3개를 추가한다.
- ICOCA 가이드를 폐기·대체하지 않는다 — 카드 있는 인원은 ICOCA(경로 A)가 더 간단(Wallet 직접
  추가)하므로 유지하고, PASMO는 카드 없는 인원용 병렬 경로로 둔다.
- 대안으로 검토했으나 채택하지 않은 것:
  - 가이드를 PASMO 중심으로 전면 교체 → 카드 있는 다수에게 PASMO 앱 경유는 불필요한 단계 추가.
  - 종이 ICOCA(경로 B)만으로 카드 미보유자 대응 → 출국 전 준비 불가·KIX 현장 동선 부담.

## Consequences (그래서)

- 긍정: Wallet 등록 카드가 없는 인원도 출국 전 모바일 IC를 준비할 수 있다. 4인 가족 IC 셋업의
  단일 실패점(카드 호환성)을 우회 경로로 해소.
- 부정·트레이드오프: PASMO 앱은 일본어 위주라 셋업 난도가 ICOCA Wallet 직접 추가보다 높다.
  무기명 특성상 분실 시 잔액 소멸. 가이드에 경로가 3개(A/B/C)로 늘어 분기가 복잡해졌다.
- 후속 행동: `data/booking-checklist.json`의 `iphone_apps` 항목(현재 "ICOCA ¥3,000/인")은
  카드 미보유자에게 PASMO 대안이 있음을 별도 PR에서 보강할 수 있다(이번 PR 범위 외).
- 영향 받은 파일: `docs/icoca-iphone-setup.md`(§1.3·§3.4·§3.5·§5·§출처).
