# 2026-06-02 — 회전초밥 비교 페이지에 저가 회전 체인(くら寿司) 추가

## Status

Accepted

## Context (왜)

- 6/2 저녁 회전초밥을 두고 사용자가 비용을 반복 검증: "(택시비 포함) 비싸잖아?" → "택시비 무관 비싸잖아" → 최종 기준 **"싸다 + 택시 + 진짜 회전"**.
- 기존 비교 2곳(大起水産·寿しのむさし)은 1인 ¥1,500~2,000으로 "회전초밥=가벼운 한 끼" 기대와 어긋남(카운터석만). 카덴쇼 도보권 すし市場는 시장 고가 + 안 도는 주문형이라 저가 카드 아님.
- 균일 저가 체인의 **회전 여부 실측 검증**: 스시로(デジロー/오토웨이터)·はま寿司는 2024~25년 회전 레인을 없애고 주문 배달형으로 전환 중 → "진짜 회전" 기준 탈락. **くら寿司만 회전 고수**(2026.3 회전 점포 수 세계 1위 기네스 인증, AI카메라+항균커버로 매일 레인 가동).
- 사용자 지시: **"비교 페이지에 추가해."**

## Decision (무엇)

- **`docs/kyoto-station-kaiten-sushi-translation.md`를 2곳→3곳 비교로 확장** — ③ 無添くら寿司 西大路七条店 추가. 한눈에 비교 표에 컬럼 추가(+1인 예산 행), 신규 섹션(점포 소개·¥115균일 메뉴 표·공식 메뉴 이미지 3장·후기·주의), 권장에 "싸게+테이블+진짜 회전 → くら寿司" 1순위 추가.
- **이미지 자가호스팅** — くら寿司 공식 메뉴 사진 3장(연어·빈토로·하마치)을 `fetch_assets.py`로 다운로드.
- **링크** — 타베로그·공식 점포·메뉴·구글맵(검색 URL) 인라인.
- **데이터 연동** — `data/itinerary.json` `places`에 `kura_sushi` 등록, 6/2 18:00 note에 "대안②(저가·테이블)"로 추가, link 라벨 3곳으로 갱신. 사람용 사본·CLAUDE.md·README도 동기화.
- **기본값 유지** — 일정 18:00 1순위는 여전히 다이키수산(산지직송 신선도·역앞 동선). くら寿司는 "가격·좌석 우선 대안"으로 비교 페이지·note에 안내.
- **검토 후 기각**: 스시로·はま寿司(회전 폐지로 "진짜 회전" 기준 미달), すし市場(시장 고가+안 도는 주문형).

## Consequences (그래서)

- 긍정: 가격(1인 ¥1,000~1,200·총액 ₩1.5~2.5만 절약)·테이블석(시부모 동반)·진짜 회전을 모두 만족하는 선택지를 가족이 현장에서 고를 수 있음. "역앞 회전초밥은 다 비싸다"는 사용자 지적에 저가 대안으로 응답.
- 부정·트레이드오프: 비교 대상 +1로 유지 부담 증가. くら寿司는 맛·신선도가 ①②보다 아래(타베로그 3.0대 초반)이고 역앞이 아니라 택시 필요(단 5분·저렴). 페이지에 명시.
- 영향 받은 파일: `docs/kyoto-station-kaiten-sushi-translation.md`(③ 섹션), `scripts/build_index.py`(DOC_PAGES 제목·설명), `data/itinerary.json`(places `kura_sushi`·6/2 note·link), `docs/kyoto-itinerary-may31-jun3-2026.md`(§1.3), `CLAUDE.md`·`README.md`, `data/local-image-map.json`·`assets/place-images/`(+3장).
