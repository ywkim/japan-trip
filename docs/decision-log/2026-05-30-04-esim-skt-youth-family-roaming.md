# 2026-05-30 — 데이터/연결성: Airalo eSIM에서 SKT 청년 가족 로밍으로 전환

## Status

Accepted — `2026-05-26-03-esim-global-marketplace-airalo.md`(영욱 Airalo 5GB)와 소연 Airalo 3GB 추가분(2026-05-28)을 Superseded.

## Context (왜)

- 부부(영욱·소연) 모두 SKT, 시부모는 KT. 직전 결정은 영욱 Airalo Moshi Moshi 5GB ₩13,800 + 소연 Airalo 3GB ₩11,040 + 시부모 핫스팟 ₩0 = **4인 합계 ₩24,840**(`data/booking-checklist.json` esim, status 예약중).
- 사용자가 "소연 로밍을 SKT 청년 혜택으로, 영욱도 SKT(청년 아님)" 구성을 검토 요청(2026-05-30).
- 실시간 리서치(2026-05-30):
  - SKT **0 청년**(만 34세 이하) 요금제는 baro 로밍 **상시 50% 할인**. baro 8GB(6+2 프로모션) 정상 ₩39,000 → ₩19,500. 0청년 50%는 여름 데이터증정 프로모션(2026-04-23~08-21)과 **중복 적용**(SKT 뉴스룸 224176).
  - **가족 로밍**: 대표 1명이 baro 가입 후 +₩3,000으로 가족(최대 4~5명)이 같은 데이터 풀 공유. 대표 자격은 "만 19세 미만·청소년 요금제"만 제외 → 0 청년은 대표 가능(T월드 NA00006995). 멤버는 다른 로밍 요금제 중복 가입 불가. 데이터 공유는 SKT 회선만(시부모 KT 제외, 핫스팟 테더링은 가능).
  - 가족 관계 증명: **SKT 국내 가족결합** 보유 시 T월드 온라인 간편 가입. **부부 가족결합 상태 확인됨**(2026-05-30 사용자 확인) → 전제 조건 충족.
- 실사용 추정(기존): 영욱 ~3.5GB, 소연 ~450MB, 시부모 합계 ~64MB → 4인 ~4GB, 영욱에 집중. 데이터를 거의 안 쓰는 청년(소연)이 50% 할인 대표가 되고, 실사용자(영욱)가 ₩3,000 멤버로 붙는 구조가 비용 최적.

## Decision (무엇)

- **소연(0 청년)이 가족 로밍 대표로 baro 8GB ₩19,500(50% 할인) 가입 + 영욱을 가족공유 멤버(+₩3,000)로 추가 + 시부모는 핫스팟 ₩0** → 4인 합계 **₩22,500**, 8GB 풀 2회선 공유. `data/booking-checklist.json` esim 항목을 이 구성으로 갱신(status 예약중 유지 — 가입·결제 전).
- 기각한 대안:
  - **Airalo 영욱5GB+소연3GB ₩24,840** — ₩2,340 비싸고 eSIM 2개 설치·이중유심 부담, SKT 번호 유지·에이닷 한국통화 무료·Club T 혜택 없음.
  - **영욱 단독 baro(비청년 정상가 ₩39,000)** — 가족 멤버(+₩3,000) 대신 본인 회선으로 사면 비효율. 반드시 청년(소연) 대표 + 영욱 멤버 구조여야 함.
  - **소연 baro 3GB ₩14,500 + 영욱 멤버 ₩3,000 = ₩17,500** — 더 싸지만 영욱 3.5GB 단독으로도 3GB 초과(소진 후 400kbps) → 8GB 채택.

## Consequences (그래서)

- 긍정: ₩22,500으로 직전안 대비 ₩2,340 절감 + SKT 번호 유지(OTP·통화)·에이닷 한국통화 무료·eSIM 무설치·Club T 혜택. 8GB 풀로 영욱 3.5GB + 시부모 핫스팟에도 여유.
- 부정·트레이드오프: 출국 전 가족로밍 신청 단계 추가(가족결합 기반 T월드 온라인). 데이터 풀 공유는 SKT 회선만 — 시부모는 핫스팟 의존(기존과 동일).
- 후속 행동:
  - 소연 T 앱/마이페이지에서 **첫 로밍 70% 할인**(만 20~34세·최근 36개월 로밍 미사용·1회성) 자격·적용가 확인 → 50%(₩19,500)보다 유리하면 그 값으로 갱신.
  - 가입 화면에서 청년 baro(YT) 실제 용량(보너스 포함) 확인.
  - **가입·결제 완료 시** esim 항목 `confirmed_booking` 승격 + 확정 일지(`2026-05-XX-esim-confirmed.md`) 추가.
