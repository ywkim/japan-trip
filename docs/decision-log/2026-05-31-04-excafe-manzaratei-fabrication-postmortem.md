# 2026-05-31 — eX cafe·만자라테이 번역 문서 날조·렌더 결함 포스트모템

## Status

Accepted

## Context (왜)

PR #89(6/1 eX cafe·만자라테이 일본어 리뷰 한국어 번역)에서 작성한 두 번역 문서(`docs/excafe-review-translation.md`·`docs/manzaratei-review-translation.md`)가 배포 후 모바일에서 다음 결함을 드러냈다(사용자 스크린샷·지적, 2026-05-31).

**날조·오류 (가장 심각)** — `CLAUDE.md` "실시간 가격 리서치 우선 / 출처 없는 숫자 입력 금지" 규칙 위반:
- **만자라테이를 "6/1 월요일 휴무"로 표기** — 사실과 반대. Tabelog·ぐるなび 모두 정기휴무 없는 **不定休(부정기 휴무)**. 6/1은 예약 가능.
- **만자라테이 메뉴가 날조** — "오반자이 세트 ¥6,500~8,000", "카이세키 풀코스 ¥12,000~15,000", "런치 11:00~14:00"을 출처 없이 작성. 실제 ぐるなび 예산 약 ¥4,800(연회 ¥5,500), 영업 **17:00~23:00 디너 전용**(런치 없음).
- **만자라테이 "여름 강변 야외석(川床)" 섹션 날조** — ぐるなび·Tabelog 어디에도 이 가게의 川床 기재 없음. 폰토초 일반 특성을 특정 가게 사실인 양 단정.
- **만자라테이 주소·역사 부정확** — "130년·에도 시대 목재" 등 미검증 서술, 주소도 旧표기(鍋屋町209-13).
- **eX cafe 당고 세트 ¥1,320·말차 빙수 ¥1,650 날조** — Tabelog 메뉴 실값은 **호쿠호쿠 당고 세트 ¥1,850·말차 빙수 ¥1,600**.

**렌더·UX 결함**:
- **문서 하단 깨진 마크다운 링크** — 다단계 Edit을 어설프게 수행하다 `(2026-05-31 검색)(https://www.google.com/maps/search/%...)` 형태의 생(raw) URL이 잔류. 화면에 그대로 노출.
- **카페 비교표가 5열** — 모바일에서 가로 스크롤 발생(`.doc table { overflow-x:auto }`). 비미관.
- **Wikimedia Commons 이미지가 세로로 과대** — `.doc img` 폭 제한 규칙 부재 + 사용자가 이전에도 "카페 페이지 Wikimedia 어색" 지적.
- **카페 비교표 Google Maps 검색 링크가 엉뚱한 주소로 연결** — 검색 질의 부정확, 일본어 1차 출처(Tabelog) 미확인.

**오해였던 항목 (실제로는 정상)**:
- "origin/main 카드 블로그를 계속 대체하는가?" — `data/itinerary.json`의 6/1 12:00 `blog_reviews`(15장)는 origin/main과 **바이트 동일**. 두 배포(nihon-trip / 프리뷰)는 같은 가로 카드 스트립을 다른 위치로 스크롤한 화면이었음(eX cafe 카드 ↔ 嵯峨野湯·パン&エスプ 카드는 모두 같은 스트립의 일부). 데이터 변경 없음.

## Decision (무엇)

두 문서를 **Tabelog·ぐるなび 일본어 1차 정보로만** 전면 재작성하고, 검증 불가 내용은 삭제한다. 렌더 결함은 빌드 CSS·문서 구조로 해소한다.

1. **만자라테이 재작성** — 휴무 `不定休`(정기휴무 없음, 6/1 영업 가능)로 정정 + 휴무 근거 출처 링크(ぐるなび·Tabelog) 명시. 영업 `17:00~23:00`(디너 전용), 예산 `¥4,000~5,000`, 주소 `下樵木町198-1`로 정정. 날조 메뉴·川床 섹션·미검증 역사·Wikimedia 이미지 **삭제**. 하단 깨진 링크 제거.
2. **eX cafe 재작성** — 당고 세트 `¥1,850`·말차 빙수 `¥1,600`으로 정정(Tabelog 메뉴 출처). 5열 비교표를 **카페별 리스트(가게명+특색+위치·시간+Tabelog 링크)** 로 변환 — 모바일 가로 스크롤 제거. Google Maps 검색 링크 → 각 카페 **Tabelog 페이지 링크**로 교체(% Arabica 26025913·嵯峨野湯 26005250·パンとエスプレッソと 26032681·雲ノ茶 26038471, 주소·영업시간 모두 검증).
3. **빌드 CSS** — `DOC_CSS`에 `.doc img { max-width:100%; height:auto; ... }` 추가(이미지 과대·가로 넘침 방지). TDD: `test_doc_images_constrained_to_container` 선작성.
4. **DOC_PAGES 제목·설명 정정** — "100년 목조 가옥"·"130년 전통·川床" 등 미검증 표현 제거.

채택하지 않은 대안:
- 표 CSS만 손질(열 유지) — 5열은 모바일에서 근본적으로 좁으므로 리스트 전환이 우월. 미채택.
- 문서 폐기 — 사용자가 6/1 외식 참고 자료를 원함. 검증 가능한 사실로 유지가 정답.

## Consequences (그래서)

**긍정**:
- 두 문서가 Tabelog·ぐるなび 1차 출처로 검증됨(가격·영업·휴무·주소). 모든 비교 카페에 Tabelog 링크.
- 모바일 가로 스크롤·과대 이미지·깨진 링크 제거. `.doc img` 규칙은 전 문서 페이지(카네요·신쿄고쿠 등)에 일괄 적용.
- 잘못된 "월요일 휴무" 안내로 6/1 저녁을 헛걸음할 위험 제거.

**부정·트레이드오프**:
- 문서가 짧아짐(날조 분량 삭제). 단, 길이보다 정확성이 본 레포 규칙의 우선순위.
- 川床·역사 등 "이야깃거리"는 출처 확보 전까지 보류.

**후속 행동**:
- 향후 식당·장소 문서는 **반드시 일본어 1차 출처(Tabelog/공식/ぐるなび) 확인 후** 수치 입력. "그쯤" 추정 금지(기존 규칙 재확인).
- 다단계 Edit 후에는 대상 라인을 **재독(Read)으로 검수**해 생 URL·깨진 마크다운 잔류 여부 확인.

**영향 받은 파일**:
- `docs/excafe-review-translation.md`·`docs/manzaratei-review-translation.md` (전면 재작성)
- `scripts/build_index.py` (DOC_CSS `.doc img` + DOC_PAGES 제목·설명)
- `tests/test_build_index.py` (`test_doc_images_constrained_to_container`)
- `CLAUDE.md`·`README.md` (DOC_PAGES 21→23 반영, 두 페이지 안내 — PR #89에서 누락분 보완)
