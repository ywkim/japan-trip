# 2026-06-01 — 6/1 오후: 금각사만 보고 시오 마치야 복귀 휴식

## Status

Accepted

(이전 일지 `2026-06-01-06-afternoon-restructure-heat-avoidance.md`의 "냉방 휴식(백화점·카페)" 부분을 본 결정이 대체)

## Context (왜)

직전 재정렬(`2026-06-01-06`)에서 6/1 오후 료안지를 빼고 14:40~17:00을 **백화점·카페 냉방 휴식**으로 채웠다. 그러나 사용자 피드백:

- "교토에 와서 백화점이 목적인가?" — 백화점은 여행 목적이 아님
- "eX cafe(당고 카페)가 오전에 있어 카페가 또 겹친다" — 오후 카페는 중복
- 료안지·닌나지·니조성을 실제 대안으로 검토했으나, 셋 다 실외 사찰·성이라 더위·도보 부담 잔존 (료안지조차 방장이 돌계단 위 — 입구 오르내림 왕복 10~15분 확인)

결정적 조건:
- 6/1은 **시오 마치야가 2박째 숙소**라 오후에 비어 있음 (추가 비용 0)
- 시오는 **니조역 도보 7분** = 금각사(북서)와 폰토초 저녁(도심) **사이**에 위치 → 되돌아옴 최소
- 시부모(고령자 2인)에게는 **자기 숙소에서 누워 쉬고 샤워**하는 것이 백화점·카페보다 회복에 우월

검증한 동선(실시간 리서치):
- 금각사도(金閣寺道) → 니조성앞(二条城前): **시버스 12번 약 20분·¥230·환승 0회** (京都市交通局/jorudan 2026-06-01)
- 니조성앞 → 시오 도보 약 10분
- 시오 → 폰토초: 지하철 도자이선 니조역→교토시야쿠쇼마에역 6분·¥220 + 도보

## Decision (무엇)

**6/1 오후를 "금각사만 짧게 보고 시오 마치야로 복귀해 휴식"으로 확정한다.**

- 14:00 금각사 — 일방통행 순환 30분, 포토존 한 컷
- **15:20 시오 마치야 복귀 — 냉방 휴식 약 1.5시간** (신규 일정 항목)
- 17:30 폰토초 만자라테이 저녁 (시오→도자이선→폰토초)

**채택하지 않은 대안**:
- 백화점·카페 휴식 — 여행 목적 아님 + 오전 eX cafe와 중복
- 료안지 — 방장이 돌계단 위(왕복 10~15분 오르내림) + 부분 햇볕 정원길, 완전 더위 회피 아님
- 닌나지 — 영내 넓어 도보 많고 백트래킹, 6월은 오무로 벚꽃도 없음
- 니조성 — 평지지만 성 전체 도보량 큼·¥1,300, 또 하나의 실외 일정

## Consequences (그래서)

**긍정**:
- 시부모 더위·체력 회복 극대화 (완전 냉방·샤워·누워서 휴식)
- 추가 비용 0 (이미 예약된 숙소), 짐 없이 가볍게
- 동선 효율: 시오가 금각사–도심 사이라 되돌아옴 최소
- 한낮 최고 기온 구간(15~17시)을 실내에서 완전 회피

**부정·트레이드오프**:
- 오후 관광 1곳(료안지급 사찰) 포기 — 단 본 여행의 일관된 우선순위(고령자 무리 없이)에 부합
- 시오 복귀 후 다시 폰토초로 나가는 지하철 1회(¥220/인) 추가

**후속 행동**:
- 6/1 아침 시버스 12번 금각사도 승차 위치(C乗り場) 확인
- 시오 체크인 시 오후 재입실(낮 휴식) 가능 여부는 자명(2박째 동일 숙소)

**영향 받은 파일**:
