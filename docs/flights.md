## 4. 의사결정 기준에 미치는 영향

### 4.1 비용 (`cost`, 낮을수록 좋음)
- 후쿠오카 > 오사카(GMP) ≈ 삿포로 > 도쿄(NRT) > 오키나와 ≈ 교토 > 도쿄(GMP-HND) 순으로 부담 적음
- **점수 환산 가이드** (4인 median 기준):
  - 60만 이하 → 9~10점
  - 80~110만 → 7~8점
  - 110~140만 → 5~6점
  - 170만 이상 → 3~4점

### 4.2 신체 부담 (`physical_burden`)
- 비행시간이 직접 영향: 후쿠오카 75분 < 오사카 105분 < 도쿄·오키나와 135~150분 < 삿포로 165분
- GMP 출발은 **자택→공항 이동 시간**을 줄여 시부모 동반 시 가산점 (도쿄·오사카 한정)

### 4.3 가족 적합성 (`family_fit`)
- 시부모 동반이면 **GMP 출발이 가능한 도쿄·오사카**가 동선상 유리
- 단, 도쿄(GMP-HND)는 비용 페널티가 큼 → 오사카(GMP-KIX)가 가장 균형 잡힌 선택지
