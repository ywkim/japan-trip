# 의사결정 일지

여행 결정 과정에서의 합의·보류·변경 사항을 시간순으로 기록.

## 형식

```
### YYYY-MM-DD — 주제
- 합의: ...
- 보류: ...
- 변경: ...
- 다음 단계: ...
```

## 일지

### 2026-05-09 — 레포 초기화
- 합의: 일본 여행을 정량적으로 비교하여 결정
- 합의: MCDA(가중평가) 프레임 사용 — 7개 기준, 후보 6개로 시작
- 보류: 동반자(부부만 vs +시부모), 기간, 시기, 예산
- 다음 단계: 각자 후보별 점수 입력 → 가중치 함께 조정

### 2026-05-09 — 교토 시나리오 제안 (5/24~27, 시부모 동반 4인)
- 제안: 교토 3박4일, 마지막 박 우메코지 카덴쇼 료칸 (조식 포함 플랜)
  - 항공: 에어서울 인천↔간사이 직항, 4인 ₩1,170,638
  - 료칸: 트윈 2실 (커넥팅룸 요청), 1박 ₩1,599,926 (조식 포함)
  - 조식 포함 채택 근거: 아버님이 아침 편히 드시는 걸 선호
  - 총 예상 비용 약 ₩5,486,000 (5M~7M 사이)
- 점수 입력 (교토): cost 6 / schedule_fit 8 / physical_burden 6 / experience 9 / family_fit 8 / medical_access 6 / seasonality 9
- 보류: 1·2박 시티 호텔 선정, 가이세키 저녁 위치 (료칸 내 vs 외부), 시부모 동반 최종 확정
- 다음 단계: 다른 후보지(도쿄·오사카 등) 점수 입력 후 비교 → 가중치 민감도 확인 → 최종 결정
