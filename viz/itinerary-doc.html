6/2 후시미이나리 입구(1구간만) → 도후쿠지 → 카덴쇼 일찍 체크인 → 대욕장 시부모 무리 없는 보행량
6/3 교토역 이세탄 쇼핑 → 출국

4.2 서북 사찰 집중형 — 세계유산 사찰 + 우지

금각사·료안지·닌나지·묘신지 서북 라인 + 우지 뵤도인. 총 도보 약 14km.

날짜 코스 포인트
5/31 기요미즈데라 → 산넨자카 → 야사카 신사 → 폰토초 저녁 (빡빡형과 동일 첫날)
6/1 금각사 → 료안지 → 닌나지(오무로 우동 점심) → 묘신지 → 란덴 전차 귀환 대규모 선종 사찰 집중
6/2 후시미이나리 → 우지 뵤도인(세계유산, 10엔 동전 사찰) → 우지 말차 → 카덴쇼 체크인 JR 나라선 우지역 이동
6/3 도지 오층탑 또는 이세탄 → 출국

4.3 미식+문화 체험형 — 니시키 시장·도예·폰토초

니시키 시장·교토 도예 체험·오반자이 정식·폰토초 골목 중심. 총 도보 약 12km.

날짜 코스 포인트
5/31 니시키 시장 점심 → 기요미즈데라+도자기 거리 → 야사카 신사 → 폰토초 이자카야 도착 당일 미식 집중
6/1 죽림길 → 텐류지 → 두부 가이세키 → 도예 공방 체험 → 니시키 재방문 → 오반자이 저녁 사가 도예 기념관 체험
6/2 후시미이나리 → 도지 오층탑 → 이세탄 식품관 점심 → 카덴쇼 체크인 → 우메코지 공원 교토 철도박물관 선택 가능
6/3 교토역 포르타 기념품 → 출국
